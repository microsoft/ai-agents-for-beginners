## 🌱 開始使用

本課程的各單元涵蓋構建 AI 代理人的基礎。每堂課都有自己的主題，歡迎從任何一堂開始學習！

本課程提供多語言支援。前往我們的[可用語言](../..)。

如果你是第一次使用生成式 AI 模型開發，請參閱我們的 [生成式 AI 初學者](https://aka.ms/genai-beginners) 課程，其中包含 21 節關於使用生成式 AI 的教學。

別忘了為此倉庫[加星 (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) 並 [fork 此倉庫](https://github.com/microsoft/ai-agents-for-beginners/fork) 以執行程式碼。

### 與其他學員交流、解答你的問題

如果你遇到困難或有任何關於構建 AI 代理人的問題，請加入我們在 [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) 的專用 Discord 頻道。

### 你需要準備的東西

本課程的每堂課都包含程式範例，程式碼可在 code_samples 資料夾中找到。你可以[fork 此儲存庫](https://github.com/microsoft/ai-agents-for-beginners/fork) 來建立自己的副本。  

這些練習中的程式範例使用 Microsoft Agent Framework 以及 Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - 需要 Azure 帳戶

本課程使用下列來自微軟的 AI 代理人框架與服務：

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


如需關於執行本課程程式碼的更多資訊，請參閱[課程設定](./00-course-setup/README.md)。
