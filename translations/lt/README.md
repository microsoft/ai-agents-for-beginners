## 🌱 Pradžia

Šiame kurse yra pamokos, kurios apima pagrindus, kaip kurti AI agentus. Kiekviena pamoka aptaria savo temą, tad pradėkite nuo jums patinkančios!

Šiam kursui yra daugiakalbė palaikymas. Peržiūrėkite mūsų [pasiekiamas kalbas čia](#-multi-language-support). 

Jei pirmą kartą kuriate su generatyviniu AI modeliais, žiūrėkite mūsų kursą [Generatyvinis AI pradedantiesiems](https://aka.ms/genai-beginners), kuriame yra 21 pamoka apie darbą su GenAI.

Nepamirškite [pažymėti (🌟) šią saugyklą](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ir [padaryti šaką šiai saugyklai](https://github.com/microsoft/ai-agents-for-beginners/fork), kad galėtumėte vykdyti kodą.

### Susitikite su kitais mokiniais, gaukite atsakymus į savo klausimus

Jei įstringate arba turite klausimų apie AI agentų kūrimą, prisijunkite prie mūsų skirtos Discord kanalo [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Ką reikia turėti

Kiekvienoje šio kurso pamokoje yra kodo pavyzdžiai, kuriuos rasite code_samples aplanke. Galite [padaryti šaką šiai saugyklai](https://github.com/microsoft/ai-agents-for-beginners/fork), kad sukurtumėte savo kopiją.

Šių pavyzdžių kodas naudoja Microsoft Agent Framework su Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - reikia Azure paskyros

Šiame kurse naudojami šie Microsoft AI agentų karkasai ir paslaugos:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

Kai kurie kodo pavyzdžiai taip pat palaiko alternatyvius OpenAI suderinamus tiekėjus, tokius kaip [MiniMax](https://platform.minimaxi.com/), kuris siūlo didelio konteksto modelius (iki 204K žetonų). Daugiau informacijos rasite [Kurso nustatymuose](./00-course-setup/README.md).

Daugiau informacijos apie šio kurso kodo paleidimą rasite [Kurso nustatymuose](./00-course-setup/README.md).
