> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> Tai suteiks jums viską, ko reikia kursui, ir leis atsisiųsti daug greičiau.

## 🌱 Pradžia

Šiame kurse pateiktos pamokos apima AI agentų kūrimo pagrindus. Kiekviena pamoka nagrinėja savo temą, todėl pradėkite ten, kur jums patogiausia!

Kursas palaiko kelias kalbas. Peržiūrėkite mūsų [turimas kalbas čia](#-multi-language-support). 

Jei tai jūsų pirmas kartas kuriant su generatyviais AI modeliais, peržiūrėkite mūsų [Generatyvios AI pradedantiesiems](https://aka.ms/genai-beginners) kursą, kuriame yra 21 pamoka apie kūrimą su GenAI.

Nepamirškite [įvertinti (🌟) šią saugyklą](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ir [padaryti šaką](https://github.com/microsoft/ai-agents-for-beginners/fork), kad galėtumėte paleisti kodą.

### Susitikite kitus besimokančius, gaukite atsakymus į savo klausimus

Jei sustojote ar turite klausimų apie AI agentų kūrimą, prisijunkite prie mūsų specialaus Discord kanalo [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Ko jums reikės

Kiekviena šio kurso pamoka apima kodo pavyzdžių, kuriuos rasite code_samples aplanke. Galite [padaryti šaką šią saugyklą](https://github.com/microsoft/ai-agents-for-beginners/fork) ir susikurti savo kopiją.

Šių užduočių kodo pavyzdžiai naudoja Microsoft Agent Framework su Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) – reikalinga Azure paskyra

Šiame kurse naudojami šie Microsoft AI agentų karkasai ir paslaugos:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

Kai kurie kodo pavyzdžiai taip pat palaiko alternatyvius OpenAI suderinamus tiekėjus, tokius kaip [MiniMax](https://platform.minimaxi.com/), kuris siūlo didelės apimties kontekstų modelius (iki 204K žetonų). Daugiau informacijos rasite [kurso nustatymuose](./00-course-setup/README.md).

Daugiau informacijos apie kurso kodo vykdymą žiūrėkite [kurso nustatymuose](./00-course-setup/README.md).
