## 🌱 തുടങ്ങാൻ

ഈ കോഴ്​សിൽ അലൈജന്റ് എജൻറുകൾ നിർമ്മിക്കുന്ന അടിസ്ഥാനങ്ങൾ ഉൾക്കൊള്ളുന്ന പാഠങ്ങൾ ഉണ്ട്. ഓരോ പാഠവും സ്വന്തം വിഷയം ഉൾക്കൊള്ളുന്നതുകൊണ്ട് നിങ്ങൾക്ക് ഇഷ്ടമുള്ളതു മുതൽ തുടങ്ങിയാൽ മതി!

ഈ കോഴ്സിന് ബഹുഭാഷാ പിന്തുണ ഉണ്ട്. [ഇവിടെ ലഭ്യമായ ഭാഷകൾ](#-multi-language-support) കാണുക.

ജെനറേറ്റീവ് AI മോഡലുകളുമായി ആദ്യമായി പ്രവർത്തിച്ചുകൊണ്ടിരിക്കുകയാണെങ്കിൽ, നമ്മുടെ [ജെനറേറ്റീവ് AI ഫോBeginners](https://aka.ms/genai-beginners) കോഴ്സ് പരിശോധിക്കുക, ഇത് ജെൻഎഐ ഉപയോഗിച്ച് 21 പാഠങ്ങൾ ഉൾക്കൊള്ളുന്നു.

ഈ റീപോസ്‌റ്ററി [സ്റ്റാർ (🌟) ചെയ്യാനും](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ഈ റീപോസിറ്ററി [ഫോർക്ക് ചെയ്യാനും](https://github.com/microsoft/ai-agents-for-beginners/fork) മറക്കരുത്, കോഡ് പ്രവർത്തിപ്പിക്കാൻ.

### മറ്റ് അഭ്യസകരെ കാണുക, നിങ്ങളുടെ ചോദ്യങ്ങൾക്ക് ഉത്തരങ്ങൾ നേടുക

അലൈജന്റ് എജൻറുകൾ നിർമ്മിക്കുന്നതിൽ നിങ്ങൾക്ക് പ്രശ്നം നേരിടുകയോ ചോദ്യങ്ങളുണ്ടാകുകയോ ചെയ്താൽ, ഞങ്ങളുടെ പ്രത്യേക Discord ചാനലിൽ ചേരുക [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### നിങ്ങൾക്ക് ആവശ്യമുള്ളത്

ഈ കോഴ്സിലെ ഓരോ പാഠത്തിനും കോഡ് ഉദാഹരണങ്ങൾ ഉൾക്കൊള്ളുന്നു, അവ code_samples ഫോൾഡറിൽ കിട്ടും. നിങ്ങൾക്ക് [ഇത് ഫോർക്ക് ചെയ്ത്](https://github.com/microsoft/ai-agents-for-beginners/fork) നിങ്ങളുടെ സ്വന്തം കോപ്പി ഉണ്ടാക്കാം.

ഈ പ്രായോഗിക ഉദാഹരണങ്ങൾ Microsoft Agent Framework നെ Azure AI Foundry Agent Service V2 ഉപയോഗിച്ച് പ്രവർത്തിപ്പിക്കുന്നു:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure അക്കൗണ്ട് ആവശ്യം

ഈ കോഴ്‌സിൽ Microsoft-ന്റെ താഴെപ്പെട്ട AI എജന്റ് ഫ്രെയിംവർക്ക്‌കളും സേവനങ്ങളും ഉപയോഗിക്കുന്നു:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

ചില കോഡ് ഉദാഹരണങ്ങൾ [MiniMax](https://platform.minimaxi.com/) പോലുള്ള OpenAI അനുയോജ്യമായ ഫോഴ്സ് സപ്പോർട്ട് ചെയ്യുന്നു, വലിയ കോൺടെക്‌സ്‌റ്റ് മോഡലുകൾ (204K ടോക്കണുകൾ വരെ) ലഭ്യമാക്കുന്നു. കോൺഫിഗറേഷൻ വിവരങ്ങൾക്കായി [Course Setup](./00-course-setup/README.md) കാണുക.

ഈ കോഴ്സ് ഇതിനകം പ്രവർത്തിപ്പിക്കാൻ കൂടുതൽ വിവരങ്ങൾക്ക് [Course Setup](./00-course-setup/README.md) സന്ദർശിക്കുക.
