# Github MCP സേർവർ ഉദാഹരണം

## വിവരണം

ഈ ഡെമോ Microsoft Reactor മുഖേന സംഘടിപ്പിച്ച AI ഏജന്റ്‌സ് ഹാക്കത്തോണിനായി സൃഷ്ടിക്കപ്പെട്ടതാണ്.

ഈ ഉപകരണങ്ങൾ ഉപയോക്താവിന്റെ Github റിപോസുകളിൽ ആധാരമാക്കി ഹാക്കത്തോൺ പ്രോജക്റ്റുകൾ ശുപാർശ ചെയ്യാൻ ഉപയോഗിക്കുന്നു.
ഇത് നടത്തപ്പെടുന്നത്:

1. **Github ഏജന്റ്** - Github MCP സേർവർ ഉപയോഗിച്ച് റിപോസും അവയിലെ വിവരങ്ങളും ശേഖരിക്കുന്നു.
2. **ഹാക്കത്തോൺ ഏജന്റ്** - Github ഏജന്റിന്റെ ഡാറ്റ എടുത്ത് ഉപയോക്താവ് ഉപയോഗിക്കുന്ന പ്രോജക്റ്റും ഭാഷകളും, AI ഏജന്റ്‌സ് ഹാക്കത്തോണിന്റെ പ്രോജക്റ്റ് ട്രാകുകളും അടിസ്ഥാനമാക്കി സൃഷ്ടിപരമായ ഹാക്കത്തോൺ പ്രോജക്റ്റ് ആശയങ്ങൾ സൃഷ്ടിക്കുന്നു.
3. **ഇവന്റ്‌സ് ഏജന്റ്** - ഹാക്കത്തോൺ ഏജന്റിന്റെ നിർദേശത്തിന്റെ അടിസ്ഥാനത്തിൽ, AI ഏജന്റ് ഹാക്കത്തോൺ സീരിസിൽ നിന്നുള്ള ബന്ധപ്പെട്ട ഇവന്റ്‌സ് ശുപാർശ ചെയ്യും.

## കോഡ് പ്രവർത്തിപ്പിക്കൽ

### പരിസ്ഥിതി ചിറകുകൾ

ഈ ഡെമോ Microsoft Agent Framework, Azure OpenAI Service, Github MCP Server, Azure AI Search എന്നിവ ഉപയോഗിക്കുന്നു.

ഈ ഉപകരണങ്ങൾ ഉപയോഗിക്കുന്നതിന് ആവശ്യമായ ശരിയായ പരിസ്ഥിതി ചിറകുകൾ സജ്ജീകരിച്ചിട്ടുണ്ടെന്ന് ഉറപ്പാക്കുക:

```python
AZURE_AI_PROJECT_ENDPOINT=""
AZURE_AI_MODEL_DEPLOYMENT_NAME=""
AZURE_SEARCH_SERVICE_ENDPOINT=""
AZURE_SEARCH_API_KEY=""
``` 

## Chainlit സേർവർ പ്രവർത്തിപ്പിക്കൽ

MCP സേർവർക്ക് കണക്റ്റ് ചെയ്യാൻ, ഈ ഡെമോ ചാറ്റ് ഇന്റർഫേസായി Chainlit ഉപയോഗിക്കുന്നു.

സേർവർ പ്രവർത്തിപ്പിക്കാൻ, നിങ്ങളുടെ ടെർമിനലിൽ താഴെയുള്ള കമാൻഡ് ഉപയോഗിക്കുക:

```bash
chainlit run app.py -w
```

ഇത് നിങ്ങളുടെ Chainlit സേർവർ `localhost:8000`-ൽ തുടങ്ങി, Azure AI Search ഇന്‍ഡക്സിൽ `event-descriptions.md` ഉള്ളടക്കം പൂരിപ്പിക്കും.

## MCP സേർവറുമായി കണക്റ്റ് ചെയ്യൽ

Github MCP Server-ലേക്ക് കണക്റ്റ് ചെയ്യാൻ "Type your message here.." ചാറ്റ് ബോക്‌സിന്റെ താഴെ കാണുന്ന "plug" ഐക്കൺ തിരഞ്ഞെടുക്കുക:

![MCP Connect](../../../../../translated_images/ml/mcp-chainlit-1.7ed66d648e3cfb28.webp)

അവിടെ നിന്ന് "Connect an MCP" ക്ലിക്ക് ചെയ്ത് Github MCP Server-ലേക്ക് കണക്റ്റ് ആകാനുള്ള കമാൻഡ് ചേർക്കുക:

```bash
npx -y @modelcontextprotocol/server-github --env GITHUB_PERSONAL_ACCESS_TOKEN=[YOUR PERSONAL ACCESS TOKEN]
```

"[YOUR PERSONAL ACCESS TOKEN]" നിങ്ങളുടെ യഥാർത്ഥ Personal Access Token-നാൽ മാറ്റുക.

കണക്റ്റ് ചെയ്‌തതിൽ മുൻപായി, പ്ലഗ് ഐക്കണിന് அருகിൽ (1) കാണണം, ഇത് കണക്റ്റ് ആണെന്ന് സ്ഥിരീകരിക്കുന്നു. കാണാത്തപ്പോൾ `chainlit run app.py -w` ഉപയോഗിച്ച് Chainlit സേർവർ പുനരാരംഭിക്കുക.

## ഡെമോ ഉപയോഗിക്കൽ

ഹാക്കത്തോൺ പ്രോജക്റ്റുകൾ ശുപാർശ ചെയ്യാനുള്ള ഏജന്റ് പ്രവാഹം തുടങ്ങാൻ, താങ്കൾക്ക് താഴെയുള്ള പോലുള്ള ഒരു സന്ദേശം ടൈപ്പ് ചെയ്യാം:

"Recommend hackathon projects for the Github user koreyspace"

റൂട്ടർ ഏജന്റ് നിങ്ങളുടെ അഭ്യർത്ഥന വിലയിരുത്തി ഏజന്റ്‌സ് (GitHub, Hackathon, Events) ഏത് കോമ്പിനേഷനാണ് നന്നായി നിങ്ങളുടെ ചോദ്യം കൈകാര്യം ചെയ്യാൻ യോഗ്യമായത് എന്ന് നിർണ്ണയിക്കും. Github റിപോസ് വിശകലനം, പ്രോജക്റ്റ് ആശയ നിർമ്മാണം, ബന്ധപ്പെട്ട ടെക് ഇവന്റുകൾ എന്നിവയുടെ അടിസ്ഥാനത്തിൽ ഏജന്റ്‌സ് ഒന്നിച്ചുള്ള സമഗ്ര ശുപാർശകൾ നൽകും.
