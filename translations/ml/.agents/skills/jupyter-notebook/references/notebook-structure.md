# നോട്ട്ബുക്ക് ഘടന

Jupyter നോട്ട്ബുക്കുകൾ ഈ ഉയർന്ന നിലയുടെ രൂപത്തിലുള്ള JSON ഡോക్యുമെന്റുകളാണ്:

- `nbformat` ഒപ്പം `nbformat_minor`
- `metadata`
- `cells` (Markdown-ഉം കോഡ് സെല്ലുകളുടെ പട്ടിക)

When editing `.ipynb` files programmatically:

- ടെംപ്ലേറ്റിൽ നിന്നുള്ള `nbformat` ഒപ്പം `nbformat_minor` സംരക്ഷിക്കുക.
- `cells` ഒരു ക്രമത്തിലുള്ള പട്ടികയായി നിലനിർത്തുക; ഉദ്ദേശ്യപരമായപ്പോൾ മാത്രമേ ക്രമം മാറ്റാവൂ.
- കോഡ് സെല്ലുകൾക്കായി, അറിയാത്തപ്പോൾ `execution_count`-നെ `null` ആയി സജ്ജീകരിക്കുക.
- കോഡ് സെല്ലുകൾക്കായി, scaffolding ചെയ്യുമ്പോൾ `outputs`-നെ ശൂന്യപ്പെട്ട പട്ടികയായി സജ്ജീകരിക്കുക.
- Markdown സെല്ലുകൾക്കായി, `cell_type="markdown"` ഒപ്പം `metadata={}` നിലനിർത്തുക.

Raw notebook JSON കൈകൊണ്ട് എഴുതുന്നതിനുപകരം ബണ്ടില്‍ ചെയ്ത ടെംപ്ലേറ്റുകള്‍ അല്ലെങ്കിൽ `new_notebook.py` (ഉദാഹരണത്തിന്, `$CODEX_HOME/skills/jupyter-notebook/scripts/new_notebook.py`) ഉപയോഗിച്ച് scaffolding ചെയ്യുന്നതിനെ മുൻഗണിക്കുക.
