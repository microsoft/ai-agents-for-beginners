> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'

>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"

>

## 🌱 Začetek

Ta tečaj vsebuje lekcije, ki pokrivajo osnove gradnje AI agentov. Vsaka lekcija obravnava svojo temo, zato začnite kjerkoli želite!

Ta tečaj podpira več jezikov. Obiščite našo stran z [razpoložljivimi jeziki tukaj](../..).

Če je to vaš prvič, da gradite z generativnimi AI modeli, si oglejte naš tečaj [Generativna umetna inteligenca za začetnike](https://aka.ms/genai-beginners), ki vključuje 21 lekcij o gradnji z GenAI.

Ne pozabite [ozvezditi (🌟) tega skladišča](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) in [narediti vilice tega skladišča](https://github.com/microsoft/ai-agents-for-beginners/fork), da boste lahko pognali kodo.

### Spoznajte druge učence, dobite odgovore na svoja vprašanja

Če boste naleteli na težave ali imeli vprašanja o gradnji AI agentov, se pridružite našemu namenskem Discord kanalu v [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Kaj potrebujete

Vsaka lekcija v tem tečaju vključuje primere kode, ki jih najdete v mapi code_samples. Lahko [naredite vilice tega skladišča](https://github.com/microsoft/ai-agents-for-beginners/fork), da ustvarite svojo kopijo.

Primeri kode v teh vajah uporabljajo Microsoft Agent Framework z Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - zahtevan račun Azure

Ta tečaj uporablja naslednje ogrodja in storitve AI agentov iz Microsofta:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


Za več informacij o zagonu kode za ta tečaj obiščite [namestitev tečaja](./00-course-setup/README.md).
