# AGENTS.md

## প্রকল্পের ওভারভিউ

এই রিপোজিটরিতে রয়েছে "শুরুকারীদের জন্য AI এজেন্টস" - একটি বিস্তৃত শিক্ষা কোর্স যা AI এজেন্ট তৈরি করার জন্য প্রয়োজনীয় সবকিছু শেখায়। কোর্সটি ১৫+ পাঠ নিয়ে গঠিত যা AI এজেন্টের মৌলিক ধারণা, ডিজাইন প্যাটার্ন, ফ্রেমওয়ার্ক এবং প্রোডাকশন ডিপ্লয়মেন্ট কভার করে।

**মূল প্রযুক্তিসমূহ:**
- পাইথন ৩.১২+
- ইন্টারেক্টিভ শেখার জন্য জুপিটার নোটবুক
- AI ফ্রেমওয়ার্ক: মাইক্রোসফট এজেন্ট ফ্রেমওয়ার্ক (MAF)
- আজুর AI সার্ভিস: মাইক্রোসফট ফাউন্ড্রি, আজুর AI ফাউন্ড্রি এজেন্ট সার্ভিস V2

**স্থাপত্য:**
- লেসনভিত্তিক স্ট্রাকচার (০০-১৫+ ডিরেক্টরি)
- প্রতিটি লেসনে রয়েছে: README ডকুমেন্টেশন, কোড নমুনা (জুপিটার নোটবুক), এবং ইমেজ
- স্বয়ংক্রিয় অনুবাদ ব্যবস্থা দ্বারা মাল্টি-ল্যাঙ্গুয়েজ সাপোর্ট
- প্রতিটি লেসনের জন্য একটি পাইথন নোটবুক মাইক্রোসফট এজেন্ট ফ্রেমওয়ার্ক ব্যবহার করে

## সেটআপ কমান্ডসমূহ

### পূর্বশর্তসমূহ
- পাইথন ৩.১২ বা তার উপরে
- আজুর সাবস্ক্রিপশন (আজুর AI ফাউন্ড্রির জন্য)
- আজুর CLI ইনস্টল এবং প্রমাণীকৃত (`az login`)

### প্রাথমিক সেটআপ

১. **রিপোজিটরি ক্লোন বা ফর্ক করুন:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # অথবা
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```
  
২. **পাইথনের ভার্চুয়াল এনভায়রনমেন্ট তৈরি ও সক্রিয় করুন:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # উইন্ডোজে: venv\Scripts\activate
   ```
  
৩. **ডিপেন্ডেন্সি ইনস্টল করুন:**
   ```bash
   pip install -r requirements.txt
   ```
  
৪. **এনভায়রনমেন্ট ভেরিয়েবল সেট করুন:**
   ```bash
   cp .env.example .env
   # আপনার API কী এবং এন্ডপয়েন্টগুলি সহ .env সম্পাদনা করুন
   ```
  

### প্রয়োজনীয় এনভায়রনমেন্ট ভেরিয়েবলসমূহ

**আজুর AI ফাউন্ড্রি** (আবশ্যক):
- `AZURE_AI_PROJECT_ENDPOINT` - আজুর AI ফাউন্ড্রি প্রজেক্ট এন্ডপয়েন্ট  
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - মডেল ডিপ্লয়মেন্ট নাম (যেমন, gpt-4o)

**আজুর AI সার্চ** (লেসন ০৫ - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - আজুর AI সার্চ এন্ডপয়েন্ট  
- `AZURE_SEARCH_API_KEY` - আজুর AI সার্চ API কী  

প্রমাণীকরণ: নোটবুক চালানোর আগে `az login` চালান (ব্যবহার করে `AzureCliCredential`)।
