## 🌱 শুরু করা

এই কোর্সে AI এজেন্ট তৈরি করার মূল বিষয়গুলি শেখানো হয়েছে। প্রতিটি লেসনে নিজস্ব বিষয় থাকে, তাই যেখান থেকে ইচ্ছে শুরু করুন!

এই কোর্সে বহুভাষিক সমর্থন রয়েছে। আমাদের [উপলব্ধ ভাষাগুলো এখানে দেখুন](../..)।

যদি আপনি প্রথমবার জেনেরেটিভ AI মডেল দিয়ে কাজ করতে চান, তাহলে আমাদের [Generative AI For Beginners](https://aka.ms/genai-beginners) কোর্স দেখুন, যেখানে GenAI দিয়ে কাজ করার ২১টি লেসন রয়েছে।

ভুলবেন না [এই রেপোজিটরিটি স্টার (🌟) দিন](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) এবং [ফর্ক করে কোড চালান](https://github.com/microsoft/ai-agents-for-beginners/fork)।

### অন্য শিখণোয়াদের সাথে পরিচিত হন, আপনার প্রশ্নগুলোর উত্তর পান

যদি AI এজেন্ট তৈরিতে আটকা পড়েন বা কোনো প্রশ্ন থাকে, আমাদের বিশেষ Discord চ্যানেলে যোগ দিন [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord)।

### যা প্রয়োজন

এই কোর্সের প্রতিটি লেসনে কোড উদাহরণ থাকে, যা code_samples ফোল্ডারে পাওয়া যাবে। আপনি [এই রেপোটি ফর্ক](https://github.com/microsoft/ai-agents-for-beginners/fork) করে নিজের কপি তৈরি করতে পারেন।

এই উদাহরণগুলো Microsoft Agent Framework ও Azure AI Foundry Agent Service V2 ব্যবহার করে তৈরি:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure অ্যাকাউন্ট প্রয়োজন

এই কোর্সে Microsoft এর নিচের AI Agent ফ্রেমওয়ার্ক ও সার্ভিস ব্যবহার করা হয়েছে:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

এই কোর্সের কোড চালানোর বিষয়ে আরও তথ্যের জন্য দেখুন [Course Setup](./00-course-setup/README.md)।
