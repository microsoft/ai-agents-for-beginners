> हा रिपॉझिटरीमध्ये 50+ भाषांमध्ये अनुवाद समाविष्ट आहेत ज्यामुळे डाउनलोडचा आकार मोठा वाढतो. अनुवादांशिवाय क्लोन करण्यासाठी sparse checkout वापरा:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> यामुळे तुम्हाला कोर्स पूर्ण करण्यासाठी आवश्यक सर्व गोष्टी कमी वेळेत डाउनलोड करून मिळतील.

## 🌱 सुरुवात कशी करावी

या कोर्समध्ये AI एजंट्स बनविण्याच्या मूलभूत गोष्टींची धडे आहेत. प्रत्येक धडा स्वतःच्या विषयावर आहे म्हणून तुम्हाला जिथून सुरुवात करायची असेल तिथेच सुरुवात करा!

या कोर्ससाठी बहुभाषी समर्थन उपलब्ध आहे. आमच्या [उपलब्ध भाषा येथे](../..) पहा. 

जर तुम्ही जनरेटिव्ह AI मॉडेल्ससह प्रथमच काम करत असाल, तर आमचा [Generative AI For Beginners](https://aka.ms/genai-beginners) कोर्स पहा, ज्यामध्ये GenAI सह निर्माण करण्याबद्दल 21 धडे आहेत.

हे रेपो [स्टार (🌟) द्या](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) आणि कोड चालवण्यासाठी हा रेपो [फोर्क करा](https://github.com/microsoft/ai-agents-for-beginners/fork).

### इतर शिकणाऱ्यांना भेटा, तुमचे प्रश्न विचारून उत्तर मिळवा

जर तुम्हाला अडथळा आला किंवा AI एजंट्स बनवण्याबाबत प्रश्न असतील, तर आमच्या समर्पित Discord चॅनेलमध्ये जोडा: [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### काय आवश्यक आहे

या कोर्समधील प्रत्येक धड्यात कोड उदाहरणे आहेत, जी code_samples फोल्डरमध्ये आढळतील. तुमची स्वतःची कॉपी तयार करण्यासाठी तुम्ही हा रेपो [फोर्क करू शकता](https://github.com/microsoft/ai-agents-for-beginners/fork).  

या एक्सरसाइजमधील कोड उदाहरणे Microsoft Agent Framework सह Azure AI Foundry Agent Service V2 वापरतात:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure खाते आवश्यक

हा कोर्स Microsoft कडून खालील AI Agent फ्रेमवर्क्स आणि सेवांचा वापर करतो:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


या कोर्ससाठी कोड कसा चालवायचा याबद्दल अधिक माहितीसाठी, [Course Setup](./00-course-setup/README.md) पहा.
