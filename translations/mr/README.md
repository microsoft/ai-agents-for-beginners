## 🌱 सुरुवात कशी करावी

हा कोर्स AI एजंट्स तयार करण्याच्या मूलभूत गोष्टी शिकवतो. प्रत्येक धडा वेगळ्या विषयावर आहे त्यामुळे तुम्हाला जिथून सुरुवात करायची आहे तिथून सुरुवात करा!

या कोर्ससाठी बहुभाषिक समर्थन आहे. आमच्या [उपलब्ध भाषांसाठी येथे पहा](#-multi-language-support).

जर तुम्ही Generative AI मॉडेल्स सोबत प्रथमच काम करत असाल तर आमचा [Generative AI For Beginners](https://aka.ms/genai-beginners) कोर्स पहा, ज्यामध्ये GenAI सह निर्माण करण्यासाठी २१ धडे आहेत.

हा रेपॉझिटरी [स्टार (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) आणि [फोर्क](https://github.com/microsoft/ai-agents-for-beginners/fork) करायला विसरू नका जेणेकरून तुम्ही कोड चालवू शकता.

### इतर शिकणाऱ्यांशी भेटा, तुमचे प्रश्न विचाराः

जर तुम्हाला काही अडचण आली किंवा AI एजंट्स बनवण्याबाबत प्रश्न असल्यास, आमच्या [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) मधील समर्पित Discord चॅनेलमध्ये सामील व्हा.

### काय आवश्यक आहे

या कोर्समधील प्रत्येक धड्यात कोड उदाहरणे समाविष्ट आहेत, ती code_samples फोल्डरमध्ये पाहू शकता. तुम्ही [हा रेपॉझिटरी फोर्क](https://github.com/microsoft/ai-agents-for-beginners/fork) करून स्वतःची कॉपी तयार करू शकता.  

या व्यायामातील कोडमध्ये Microsoft Agent Framework सह Azure AI Foundry Agent Service V2 चा वापर केला आहे:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure Account आवश्यक

हा कोर्स Microsoft कडून खालील AI Agent फ्रेमवर्क आणि सेवा वापरतो:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

काही कोड उदाहरणे पर्यायी OpenAI-सुसंगत प्रदात्यांना देखील समर्थन देतात जसे की [MiniMax](https://platform.minimaxi.com/), जे मोठ्या संदर्भातील मॉडेल्स (२,०४,००० टोकन्सपर्यंत) ऑफर करतात. कॉन्फिगरेशन तपशीलांसाठी [Course Setup](./00-course-setup/README.md) पहा.

या कोर्ससाठी कोड चालवण्याबाबत अधिक माहितीसाठी [Course Setup](./00-course-setup/README.md) वर जा.
