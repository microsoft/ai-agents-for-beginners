# Github MCP सर्व्हर उदाहरण

## वर्णन

हे Microsoft Reactor द्वारे आयोजित AI एजंट्स हॅकथॉनसाठी तयार केलेले डेमो होते.

हे साधन वापरकर्त्याच्या Github रेपॉजिटरीजच्या आधारे हॅकथॉन प्रोजेक्ट्सची शिफारस करण्यासाठी वापरले जाते.
हे असे केले जाते:

1. **Github एजंट** - Github MCP सर्व्हर वापरून रेपॉजिटरीज आणि त्या रेपॉजिटरीजची माहिती मिळविणे.
2. **हॅकथॉन एजंट** - Github एजंटकडून डेटा घेऊन वापरकर्त्याच्या प्रोजेक्ट्स, वापरलेल्या भाषांनुसार आणि AI एजंट्स हॅकथॉनसाठी प्रोजेक्ट ट्रॅक्सवर आधारित सर्जनशील हॅकथॉन प्रोजेक्ट कल्पना मांडणे.
3. **इव्हेंट्स एजंट** - हॅकथॉन एजंटच्या सूचनेनुसार, इव्हेंट्स एजंट AI एजंट हॅकथॉन सिरीजमधील संबंधित कार्यक्रमांची शिफारस करेल.

## कोड चालविणे

### पर्यावरण चल

हे डेमो Microsoft Agent Framework, Azure OpenAI Service, Github MCP Server आणि Azure AI Search वापरते.

हे साधन वापरण्यासाठी योग्य पर्यावरण चल सेट केलेले आहेत याची खात्री करा:

```python
AZURE_AI_PROJECT_ENDPOINT=""
AZURE_AI_MODEL_DEPLOYMENT_NAME=""
AZURE_SEARCH_SERVICE_ENDPOINT=""
AZURE_SEARCH_API_KEY=""
``` 


## Chainlit सर्व्हर चालविणे

MCP सर्व्हरशी कनेक्ट होण्यासाठी, हे डेमो चॅट इंटरफेस म्हणून Chainlit वापरते.

तुमच्या टर्मिनलमध्ये खालील कमांड वापरा सर्व्हर चालविण्यासाठी:

```bash
chainlit run app.py -w
```


हे तुमचा Chainlit सर्व्हर `localhost:8000` वर सुरू करेल तसेच `event-descriptions.md` मधील कंटेंट आपल्या Azure AI Search इंडेक्समध्ये भरेल.

## MCP सर्व्हरशी कनेक्ट करणे

Github MCP सर्व्हरशी कनेक्ट होण्यासाठी, "Type your message here.." चॅट बॉक्सच्या खालील "plug" आयकॉनवर क्लिक करा:

![MCP Connect](../../../../../translated_images/mr/mcp-chainlit-1.7ed66d648e3cfb28.webp)

त्यानंतर "Connect an MCP" वर क्लिक करून Github MCP सर्व्हरशी कनेक्ट होण्याची कमांड जोडा:

```bash
npx -y @modelcontextprotocol/server-github --env GITHUB_PERSONAL_ACCESS_TOKEN=[YOUR PERSONAL ACCESS TOKEN]
```


"[YOUR PERSONAL ACCESS TOKEN]" च्या ठिकाणी तुमचा खरा वैयक्तिक प्रवेश टोकन टाका.

कनेक्ट केल्यानंतर प्लग आयकॉनच्या समोरील (1) दिसेल जे कनेक्शनची पुष्टी करेल. दिसत नसेल तर `chainlit run app.py -w` कमांडने Chainlit सर्व्हर पुनः सुरू करण्याचा प्रयत्न करा.

## डेमो वापरणे

हॅकथॉन प्रोजेक्ट्सची शिफारस करण्यासाठी एजंट वर्कफ्लो सुरू करण्यासाठी, तुम्ही असे संदेश टाइप करू शकता:

"Recommend hackathon projects for the Github user koreyspace"

राउटर एजंट तुमचा विनंती विश्लेषित करून कोणत्या एजंट्सच्या संयुक्त कार्याने (GitHub, Hackathon, आणि Events) तुमच्या प्रश्नासाठी सर्वोत्तम उत्तरे देता येतील हे ठरवेल. एजंट्स GitHub रेपॉजिटरी विश्लेषण, प्रोजेक्ट कल्पना आणि संबंधित तंत्रज्ञान कार्यक्रमांच्या आधारे पूर्ण शिफारसी प्रदान करण्यासाठी एकत्र काम करतात.
