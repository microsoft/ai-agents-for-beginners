> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git

## 🌱 Začínáme

Tento kurz obsahuje lekce pokrývající základy tvorby AI agentů. Každá lekce se věnuje vlastnímu tématu, začněte tedy, kde chcete!

Kurz podporuje více jazyků. Podívejte se na naše [dostupné jazyky zde](../..). 

Pokud stavíte s generativními AI modely poprvé, podívejte se na náš kurz [Generative AI For Beginners](https://aka.ms/genai-beginners), který obsahuje 21 lekcí o tvorbě s GenAI.

Nezapomeňte [označit tento repozitář hvězdičkou (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) a [forknout tento repozitář](https://github.com/microsoft/ai-agents-for-beginners/fork), abyste mohli spouštět kód.

### Poznejte ostatní studenty, získejte odpovědi na své otázky

Pokud narazíte na potíže nebo máte otázky k tvorbě AI agentů, připojte se na náš vyhrazený Discord kanál v [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Co potřebujete

Každá lekce v tomto kurzu obsahuje příklady kódu, které naleznete ve složce code_samples. Můžete si [forknout tento repozitář](https://github.com/microsoft/ai-agents-for-beginners/fork) a vytvořit si tak vlastní kopii.  

Příklady kódu v těchto cvičeních využívají Microsoft Agent Framework s Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) – Vyžaduje Azure účet

Tento kurz používá následující AI Agent frameworky a služby od Microsoftu:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


Pro více informací o spuštění kódu ke kurzu přejděte na [Course Setup](./00-course-setup/README.md).
