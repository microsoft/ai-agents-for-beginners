# مثال خادم Github MCP

## الوصف

كانت هذه تجربة توضيحية أُنشئت لمسابقة هاكاثون وكلاء الذكاء الاصطناعي التي استضافها Microsoft Reactor.

تُستخدم هذه الأدوات لتقديم توصيات بمشاريع الهاكاثون بناءً على مستودعات Github الخاصة بالمستخدم.
يتم ذلك عبر:

1. **Github Agent** - استخدام خادم Github MCP لاسترداد المستودعات والمعلومات حول تلك المستودعات.
2. **Hackathon Agent** - يأخذ البيانات من Github Agent ويبتكر أفكارًا إبداعية لمشاريع الهاكاثون بناءً على المشاريع واللغات التي يستخدمها المستخدم ومسارات المشاريع لهاكاثون وكلاء الذكاء الاصطناعي.
3. **Events Agent** - بناءً على اقتراحات وكيل الهاكاثون، سيقترح وكيل الفعاليات فعاليات ذات صلة من سلسلة هاكاثون وكلاء الذكاء الاصطناعي.
## تشغيل الشفرة 

### متغيرات البيئة

تستخدم هذه العرض التوضيحي Microsoft Agent Framework وخدمة Azure OpenAI وخادم Github MCP وخدمة Azure AI Search.

تأكد من إعداد متغيرات البيئة المناسبة لاستخدام هذه الأدوات:

```python
AZURE_AI_PROJECT_ENDPOINT=""
AZURE_AI_MODEL_DEPLOYMENT_NAME=""
AZURE_SEARCH_SERVICE_ENDPOINT=""
AZURE_SEARCH_API_KEY=""
``` 

## تشغيل خادم Chainlit

للاتصال بخادم MCP، يستخدم هذا العرض التوضيحي Chainlit كواجهة دردشة. 

لتشغيل الخادم، استخدم الأمر التالي في الطرفية الخاصة بك:

```bash
chainlit run app.py -w
```

من المفترض أن يبدأ هذا خادم Chainlit الخاص بك على `localhost:8000` بالإضافة إلى ملء فهرس Azure AI Search الخاص بك بمحتوى `event-descriptions.md`. 

## الاتصال بخادم MCP

للاتصال بخادم Github MCP، اختر أيقونة "القابس" الموجودة أسفل مربع الدردشة "Type your message here.." :

![اتصال MCP](../../../../../translated_images/ar/mcp-chainlit-1.7ed66d648e3cfb28.webp)

من هناك يمكنك النقر على "الاتصال بـ MCP" لإضافة الأمر للاتصال بخادم Github MCP:

```bash
npx -y @modelcontextprotocol/server-github --env GITHUB_PERSONAL_ACCESS_TOKEN=[YOUR PERSONAL ACCESS TOKEN]
```

استبدل "[YOUR PERSONAL ACCESS TOKEN]" برمز الوصول الشخصي الفعلي الخاص بك. 

بعد الاتصال، يجب أن ترى (1) بجوار أيقونة القابس لتأكيد أنه متصل. إذا لم يكن كذلك، جرّب إعادة تشغيل خادم chainlit باستخدام `chainlit run app.py -w`.

## استخدام العرض التوضيحي 

لبدء سير عمل الوكيل لتوصية بمشاريع الهاكاثون، يمكنك كتابة رسالة مثل: 

"اقترح مشاريع هاكاثون لمستخدم Github koreyspace"

سيقوم وكيل التوجيه بتحليل طلبك وتحديد أي مزيج من الوكلاء (GitHub, Hackathon, and Events) الأنسب لمعالجة استفسارك. تعمل الوكلاء معًا لتقديم توصيات شاملة بناءً على تحليل مستودعات GitHub، وتوليد أفكار للمشاريع، والفعاليات التقنية ذات الصلة.
