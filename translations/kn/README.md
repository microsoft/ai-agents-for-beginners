## 🌱 ಪ್ರಾರಂಭಿಸುವುದು

ಈ ಕೋರ್ಸ್‌ನಲ್ಲಿ AI ಏಜೆಂಟ್‌ಗಳನ್ನು ನಿರ್ಮಿಸುವ ಮೂಲಭೂತ ವಿಷಯಗಳನ್ನು ಒಳಗೊಂಡ ಪಾಠಗಳು ಇವೆ. ಪ್ರತಿ ಪಾಠವು ತನ್ನದೇ ವಿಷಯವನ್ನು ಒಳಗೊಂಡಿದೆ ಆದಕಾರಣ ನೀವು ಇಚ್ಛಿಸುವ ಎಲ್ಲಿಂದ ಪ್ರಾರಂಭಿಸಬಹುದು!

ಈ ಕೋರ್ಸ್‌ಗೆ ಬಹುಭಾಷಾ ಬೆಂಬಲ ಸಿಗುತ್ತದೆ. ನಮ್ಮ [ಲಭ್ಯವಿರುವ ಭಾಷೆಗಳು ಇಲ್ಲಿವೆ](#-multi-language-support).

ನೀವು ಮೊದಲ ಬಾರಿಗೆ ಜನರೇಟಿವ್ AI ಮಾದರಿಗಳೊಂದಿಗೆ ನಿರ್ಮಿಸುತ್ತಿದ್ದರೆ, ನಮ್ಮ [Arambhikartagaagi Generative AI](https://aka.ms/genai-beginners) ಕೋರ್ಸ್ ನೋಡಿರಿ, ಇದರಲ್ಲಿ GenAI-ನೊಂದಿಗೆ ನಿರ್ಮಿಸುವ 21 ಪಾಠಗಳು ಸೇರಿವೆ.

ಈ ರೆಪೊವನ್ನು [ನಕ್ಷತ್ರ (🌟) ನೀಡುವುದು](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ಹಾಗೂ [ನಕಲು ಮಾಡಲು](https://github.com/microsoft/ai-agents-for-beginners/fork) ಮರೆಯಬೇಡಿ.

### ಇತರೆ ಕಲಿಯುವವರನ್ನು ಭೇಟಿ ಮಾಡಿ, ನಿಮ್ಮ ಪ್ರಶ್ನೆಗಳಿಗೆ ಉತ್ತರ ದೊರಕಿಸಿ

ನೀವು ಸಿಲುಕಿದಲ್ಲಿ ಅಥವಾ AI ಏಜೆಂಟ್‌ಗಳು ನಿರ್ಮಿಸುವ ಬಗ್ಗೆ ಯಾವುದೇ ಪ್ರಶ್ನೆಗಳಿದ್ದರೆ, ನಮ್ಮ Microsoft Foundry Discord ನಲ್ಲಿ ಸಮರ್ಪಿತ Discord ಚಾನೆಲ್‍ಗೆ ಸೇರ್ಪಡೆವಾಗಿರಿ: [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### ನಿಮಗೆ ಬೇಕಾಗುವ ಕೆಲವು bagay

ಈ ಕೋರ್ಸ್‌ನ ಪ್ರತಿ ಪಾಠದಲ್ಲಿ ಕೋಡ್ ಉದಾಹರಣೆಗಳಿವೆ, ಅವುಗಳನ್ನ يمكنك كود_ಸಂಪುಟ ಎಂಬ ಫೋಲ್ಡರ್‌ನಲ್ಲಿ ಕಂಡುಹಿಡಿಯಬಹುದು. ನೀವು ನಿಮ್ಮ ಪ್ರತಿಯನ್ನು ರಚಿಸಲು [ಈ ರೆಪೊವನ್ನು ನಕಲಿಸಬಹುದು](https://github.com/microsoft/ai-agents-for-beginners/fork).

ಈ ವ್ಯಾಯಾಮಗಳಲ್ಲಿ ಕೋಡ್ ಉದಾಹರಣೆಗಳು Microsoft Agent Framework ಅನ್ನು ಬಳಸಿ Azure AI Foundry Agent Service V2-ನೊಂದಿಗೆ ಕಾರ್ಯನಿರ್ವಹಿಸುತ್ತವೆ:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure ಖಾತೆ ಅಗತ್ಯವೆ

ಈ ಕೋರ್ಸ್ Microsoft ನಿಂದ ಕೆಳಗಿನ AI ಏಜೆಂಟ್ ಫ್ರೇಮ್ವರ್ಕ್‌ಗಳು ಮತ್ತು ಸೇವೆಗಳನ್ನು ಬಳಸುತ್ತದೆ:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

ಕೆಲವು ಕೋಡ್ ಉದಾಹರಣೆಗಳು OpenAI-ಸಂಗತ ಪ್ರೊವೈಡರ್‌ಗಳಾದ [MiniMax](https://platform.minimaxi.com/) ಅನ್ನು ಸಹ समर्थन ಮಾಡುತ್ತವೆ, ಇದು ದೊಡ್ಡ ವಾಹಕ ಮಾದರಿಗಳನ್ನು (204K ಟೋಕನ್ಗಳು ವರೆಗೆ) ಒದಗಿಸುತ್ತದೆ. ಸಂರಚನಾ ವಿವರಗಳಿಗೆ ದಯವಿಟ್ಟು [ಕೋರ್ಸ್ ಸೆಟ್ ಅಪ್](./00-course-setup/README.md) ನೋಡಿ.

ಈ ಕೋರ್ಸ್‌ನ ಕೋಡ್ ನಡೆಸುವ ಬಗ್ಗೆ ಹೆಚ್ಚಿನ ಮಾಹಿತಿಗಾಗಿ [ಕೋರ್ಸ್ ಸೆಟ್ ಅಪ್](./00-course-setup/README.md) ಗೆ ತೆರಳಿ.
