## 🌱 ಆರಂಭಿಸಲು

ಈ ಕೋರ್ಸ್ AI ಏಜೆಂಟ್ಸ್ ನಿರ್ಮಿಸುವ ಮೂಲತತ್ತ್ವಗಳನ್ನು ಆವರಿಸುತ್ತದೆ. ಪ್ರತಿ ಪಾಠವು ತನ್ನದೇ ವಿಷಯವನ್ನು ಒಳಗೊಂಡಿದ್ದು, ನೀವು ಇಚ್ಛಿಸಿದ ಎಲ್ಲಿಗೆ ಪ್ರಾರಂಭಿಸಬಹುದು!

ಈ ಕೋರ್ಸ್ ಗೆ ಬಹುಭಾಷಾ ಬೆಂಬಲವೂ ಇದೆ. ನಮ್ಮ [ಲಭ್ಯವಿರುವ ಭಾಷೆಗಳು ಇಲ್ಲಿ ನೋಡಿರಿ](../..).

ನೀವು ಜನರೇಟಿವ್ AI ಮಾದರಿಗಳೊಂದಿಗೆ ಮೊದಲ ಬಾರಿಗೆ ನಿರ್ಮಿಸುತ್ತಿದ್ದರೆ, ನಮ್ಮ [ಜನರೇಟಿವ್ AI ಪ್ರಾರಂಭಿಕರಿಗಾಗಿ](https://aka.ms/genai-beginners) ಕೋರ್ಸ್ ನೋಡಿರಿ, ಅದರಲ್ಲಿ GenAI ಜೊತೆ ನಿರ್ಮಾಣಮಾಡುವ 21 ಪಾಠಗಳಿವೆ.

ಈ ರೆಪೋವನ್ನು [ನಕ್ಷತ್ರ (🌟)ಮಾಡು](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ಮತ್ತು [ಫೋರ್ಕ್ ಮಾಡಿ](https://github.com/microsoft/ai-agents-for-beginners/fork) ಕೋಡ್ ಚಲಿಸಲು ಮರೆಯಬೇಡಿ.

### ಇತರ ಕಲಿಯುವವರನ್ನು ಭೇಟಿ ಮಾಡಿ, ನಿಮ್ಮ ಪ್ರಶ್ನೆಗಳಿಗೆ ಉತ್ತರ ಪಡೆಯಿರಿ

ನೀವು ಎಲ್ಲಿ ಅಡಚಣೆ ಎದುರಿಸುತ್ತೀರಾ ಅಥವಾ AI ಏಜೆಂಟ್ಸ್ ನಿರ್ಮಿಸುವ ಕುರಿತು ಪ್ರಶ್ನೆಗಳಿದ್ದರೆ, ನಮ್ಮ ನಿಶ್ಚಿತ ಡಿಸ್ಕೋರ್ಡ್ ಚಾನೆಲ್ ನಲ್ಲಿ ಸೇರಿ [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### ಬೇಕಾಗಿರುವದರ ಕುರಿತು

ಈ ಕೋರ್ಸ್‌ನ ಪ್ರತಿ ಪಾಠದಲ್ಲಿ ಕೌಡ್ ಉದಾಹರಣೆಗಳಿವೆ, ಅವುಗಳನ್ನು code_samples ಫೋಲ್ಡರ್‌ನಲ್ಲಿ ಕಾಣಬಹುದು. ನೀವು [ಈ ರೆಪೋವನ್ನು ಫೋರ್ಕ್ ಮಾಡಿ](https://github.com/microsoft/ai-agents-for-beginners/fork) ನಿಮ್ಮ ಸ್ವಂತ ನಕಲನ್ನು ರಚಿಸಬಹುದು.

ಈ ವ್ಯಾಯಾಮಗಳಲ್ಲಿ ಬಳಕೆಯಾದ ಕೋಡ್ ಉದಾಹರಣೆಗಳು Microsoft Agent Framework ನೊಂದಿಗೆ Azure AI Foundry Agent Service V2 ಅನ್ನು ಉಪಯೋಗಿಸುತ್ತವೆ:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure ಖಾತೆ ಅಗತ್ಯವಿದೆ

ಈ ಕೋರ್ಸ್ Microsoft ನಿಂದ ಕೆಳಕಂಡ AI ಏಜೆಂಟ್ ಫ್ರೇಮ್ವರ್ಕ್‌ಗಳು ಮತ್ತು ಸೇವೆಗಳನ್ನು ಬಳಸುತ್ತದೆ:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


ಈ ಕೋರ್ಸ್‌ನ ಕೋಡ್ ಚಲಿಸುವ ಕುರಿತು ಹೆಚ್ಚಿನ ಮಾಹಿತಿಗೆ, [ಕೋರ್ಸ್ ಸೆಟಪ್](./00-course-setup/README.md) ಗೆ ಹೋಗಿ.
