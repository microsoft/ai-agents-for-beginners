>

> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (ವಿಂಡೋಸ್):**  
> ```cmd

## 🌱 ಆರಂಭಿಸಿ

ಈ ಕೋರ್ಸ್ AI ಏಜೆಂಟ್ಸ್ ನಿರ್ಮಾಣದ ಮೂಲಭೂತ ವಿಷಯಗಳನ್ನು ಒಳಗೊಂಡ ಪಾಠಗಳನ್ನು ಹೊಂದಿದ್ದು, ಪ್ರತಿ ಪಾಠವು ತನ್ನದೇ ವಿಷಯವನ್ನು ಕವರ್ ಮಾಡುತ್ತದೆ, ಆದ್ದರಿಂದ ನೀವು ಬಯಸಿದ ಎಲ್ಲಿಯಲ್ಲಾದರೂ ಪ್ರಾರಂಭಿಸಬಹುದು!

ಈ ಕೋರ್ಸ್‌ಗೆ ಬಹುಭಾಷಾ ಬೆಂಬಲ ಇದೆ. ನಮ್ಮ [ಲಭ್ಯವಿರುವ ಭಾಷೆಗಳು ಇಲ್ಲಿ ನೋಡಿ](#-multi-language-support).

ನೀವು ಮೊದಲಬಾರಿಗೆ ಜನರೇಟಿವ್ AI ಮಾದರಿಗಳೊಂದಿಗೆ ನಿರ್ಮಿಸುತ್ತಿದ್ದರೆ, ನಮ್ಮ [ಜನರೇಟಿವ್ AI ಶುರುಗಾರರಿಗೆ](https://aka.ms/genai-beginners) ಕೋರ್ಸ್ ನೋಡಿ, ಅದರಲ್ಲಿ 21 ಪಾಠಗಳು GenAI ಉಪಯೋಗಿಸುವ ಅಭ್ಯಾಸಗಳನ್ನು ಒಳಗೊಂಡಿವೆ.

ಈ ರಿಪೋಜિટರಿಯನ್ನು [ಸ್ಟಾರ್ಟ್ (🌟) ಮಾಡಬೇಕೆಂದು ಮರೆಯಬೇಡಿ](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ಮತ್ತು [ಫೋರ್ಕ್ ಮಾಡಿ](https://github.com/microsoft/ai-agents-for-beginners/fork) ಆ ಕೋಡ್ ಅನ್ನು ಚಾಲನೆ ಮಾಡಲು.

### ಇತರ ಶಿಕ్షಾರ್ಥಿಗಳನ್ನು ಭೇಟಿಯಾಗಿರಿ, ನಿಮ್ಮ ಪ್ರಶ್ನೆಗಳಿಗೆ ಉತ್ತರ ಪಡೆಯಿರಿ

ನೀವು AI ಏಜೆಂಟ್ಸ್ ನಿರ್ಮಾಣದ ಬಗ್ಗೆ ಅಡೆತಡೆಗೊಳ್ಳುತ್ತಿದ್ದರೆ ಅಥವಾ ಯಾವುದೇ ಪ್ರಶ್ನೆಗಳು ಇದ್ದರೆ, ನಮ್ಮ ನಿಗದಿತ Discord ಚಾನೆಲ್‌ಕ್ಕೆ [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) ನಲ್ಲಿ ಸೇರಿ.

### ನಿಮಗೆ ಬೇಕಾಗಿರುವ ವಸ್ತುಗಳು

ಈ ಕೋರ್ಸ್‌ನ ಪ್ರತಿ ಪಾಠವೂ ಕೋಡ್ ಉದಾಹರಣೆಗಳನ್ನು ಒಳಗೊಂಡಿರುತ್ತದೆ, ಅವುಗಳನ್ನು code_samples ಫೋಲ್ಡರ್‌ನಲ್ಲಿ ಕಾಣಬಹುದು. ನೀವು ಸ್ವಂತ ಪ್ರತಿಯನ್ನು ರಚಿಸಲು [ಫೋರ್ಕ್ ಮಾಡಬಹುದು](https://github.com/microsoft/ai-agents-for-beginners/fork).

ಈ ಅಭ್ಯಾಸಗಳಲ್ಲಿನ ಕೋಡ್ ಉದಾಹರಣೆಗಳು Microsoft Agent Framework ಜೊತೆಗೆ Azure AI Foundry Agent Service V2 ನ್ನು ಬಳಕೆಮಾಡುತ್ತವೆ:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure ಖಾತೆ ಅಗತ್ಯವಿದೆ

ಈ ಕೋರ್ಸ್‌ನಲ್ಲಿ Microsoft ನಿಂದ ಈ ಕೆಳಗಿನ AI ಏಜೆಂಟ್ ಫ್ರೇಮ್‌ವರ್ಕ್‌ಗಳು ಮತ್ತು ಸೇವೆಗಳನ್ನೂ ಬಳಸಲಾಗಿದೆ:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

ಕೆಲವು ಕೋಡ್ ಉದಾಹರಣೆಗಳು ಪರ್ಯಾಯ OpenAI-ಉಪಯುಕ್ತ ಸೇವಾದಾರರಿಗೆ ಸಹ ಬೆಂಬಲ ನೀಡುತ್ತವೆ, ಉದಾಹರಣೆಗೆ [MiniMax](https://platform.minimaxi.com/), ಇದು ಬೃಹತ್ ಸಾಂದರ್ಭಿಕ ಮಾದರಿಗಳನ್ನು (204K ಟೋಕನ್‌ಗಳವರೆಗೆ) ನೀಡುತ್ತದೆ. ಸಂರಚನೆ ವಿವರಗಳಿಗಾಗಿ [ಕೋರ್ಸ್ ಸೆಟಪ್](./00-course-setup/README.md) ನೋಡಿ.

ಈ ಕೋರ್ಸ್‌ಗೆ ಸಂಬಂಧಿಸಿದಂತೆ ಕೋಡ್ ನಡೆಸುವ ಬಗ್ಗೆ ಹೆಚ್ಚಿನ ಮಾಹಿತಿಗಾಗಿ [ಕೋರ್ಸ್ ಸೆಟಪ್](./00-course-setup/README.md) ಗೆ ಭೇಟಿ ನೀಡಿ.
