# ಗುಣಮಟ್ಟ ಪರಿಶೀಲನೆ ಪಟ್ಟಿ

- ಮೇಲಿನಿಂದ ಕೆಳಗೆ ಕನಿಷ್ಠ ಒಂದು ಬಾರಿ ಸಂಪೂರ್ಣವಾಗಿ ಚಲಾಯಿಸಿ (ಅಥವಾ ಪರಿಸರ ಅನುಮತಿಸುವಷ್ಟು).
- ಪ್ರಾರಂಭಿಕ ಸೆಲ್‌ಗಳು ಎಲ್ಲಾ ಅಗತ್ಯ ಸ್ಥಿತಿಗಳನ್ನು ಹೊಂದಿವೆ ಎಂದು ಖಚಿತಪಡಿಸಿಕೊಳ್ಳಿ; ಹಿಂದಿನ ಚಾಲನೆಗಳಿಂದ ಇರುವ ಗುಪ್ತ ಸ್ಥಿತಿಯನ್ನು ತಪ್ಪಿಸಿ.
- ಔಟ್‌ಪುಟ್‌ಗಳನ್ನು ವ್ಯವಸ್ಥಿತವಾಗಿ ಇಡಿ. ಸಣ್ಣ ಸಾರಾಂಶ ಸಾಕಾಗುವಾಗ ಭಾರೀ ಔಟ್‌ಪುಟ್‌ಗಳನ್ನು ತಪ್ಪಿಸಿ.
- ಚಿಕ್ಕ ಟೇಬಲ್‌ಗಳು, ಪ್ರಮುಖ ಮೆಟ್ರಿಕ್‌ಗಳು ಅಥವಾ ಸಂಕ್ಷಿಪ್ತ ಪ್ರಿಂಟ್‌ಔಟ್‌ಗಳನ್ನು ಆದ್ಯತೆ ನೀಡಿ.
- ಕಥನವನ್ನು ಪಾಸು ಮಾಡಬಹುದಾಗಿರಿಸುವಂತೆ ಇಡಿ. ಶೀರ್ಷಿಕೆಗಳು ಮತ್ತು ಚುಟುಕು ಬುಲೆಟ್‌ಗಳನ್ನು ಬಳಸಿ, ದೀರ್ಘ ಪ್ಯಾರಾಗ್ರಾಫ್‌ಗಳನ್ನು ತಪ್ಪಿಸಿ.
- ಅಗತ್ಯವಿದ್ದಾಗ ಮಾತ್ರ ಸಹಾಯಕ TODOಗಳನ್ನು ಬಿಡಿ, ಮತ್ತು ಅವನ್ನು ಸ್ಪಷ್ಟವಾಗಿ ಲೇಬಲ್ ಮಾಡಿ.
- ಚಾಲನೆ ಸಾಧ್ಯವಿಲ್ಲದಿದ್ದರೆ, ಅಪಾಯವನ್ನು ಗುರುತಿಸಿ ಮತ್ತು ಸ್ಥಳೀಯವಾಗಿ ಅದನ್ನು ಹೇಗೆ ಮಾನ್ಯಗೊಳಿಸಬೇಕೆಂದು ವಿವರಿಸಿ.
