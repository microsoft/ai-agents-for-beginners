# ನೋಟ್ಬುಕ್ ರಚನೆ

Jupyter ನೋಟ್‌ಬುಕ್‌ಗಳು ಈ ಉನ್ನತ ಮಟ್ಟದ ರೂಪದೊಂದಿಗೆ JSON ದಾಖಲೆಗಳಾಗಿವೆ:

- `nbformat` and `nbformat_minor`
- `metadata`
- `cells` (markdown ಮತ್ತು ಕೋಡ್ ಸೆಲ್‌ಗಳ ಪಟ್ಟಿಯಾಗಿದೆ)

When editing `.ipynb` files programmatically:

- ಟೆಂಪ್ಲೇಟಿನಿಂದ `nbformat` ಮತ್ತು `nbformat_minor` ಅನ್ನು ಕಾಯ್ದಿರಿಸಿ.
- `cells` ಅನ್ನು ಕ್ರಮಿತ ಪಟ್ಟಿಯಾಗಿ ಇಡಿರಿ; ಉದ್ದೇಶವಿಲ್ಲದೆ ಮರುಕ್ರಮಿಸಬೇಡಿ.
- ಕೋಡ್ ಸೆಲ್‌ಗಳಿಗಾಗಿ, `execution_count` ಅನ್ನು ತಿಳಿಯದಿದ್ದಾಗ `null` ಆಗಿ ಇಡಿ.
- ಕೋಡ್ ಸೆಲ್‌ಗಳಿಗಾಗಿ, ರಚನಾ ಮಾದರಿ ಸೃಷ್ಟಿಸುವಾಗ `outputs` ಅನ್ನು ಖಾಲಿ ಪಟ್ಟಿಯಾಗಿ ಇಡಿ.
- ಮಾರ್ಕ್‍ಡೌನ್ ಸೆಲ್‌ಗಳಿಗಾಗಿ, `cell_type="markdown"` ಮತ್ತು `metadata={}` ಅನ್ನು ಇಡಿರಿ.

ಕೈಯಿಂದ ಕಚ್ಚಾ ನೋಟ್‌ಬುಕ್ JSON ಅನ್ನು ರಚಿಸುವ ಬದಲಾಗಿ, ಒಟ್ಟಿಗೆ ಬಂದಿರುವ ಟೆಂಪ್ಲೇಟ್‌ಗಳ ಅಥವಾ `new_notebook.py` (ಉದಾಹರಣೆಗೆ, `$CODEX_HOME/skills/jupyter-notebook/scripts/new_notebook.py`) ನಿಂದ ರಚಿಸಲು ಪ್ರಾಧಾನ್ಯತೆ ನೀಡಿ.
