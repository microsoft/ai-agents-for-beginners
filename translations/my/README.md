## 🌱 စတင်ရန်

ဒီသင်တန်းမှာ AI Agents တည်ဆောက်ခြင်း၏ အခြေခံ မူကြမ်းများကို ဖော်ပြထားသော သင်ခန်းစာများ ပါဝင်သည်။ သင်ချင်သည့် အပိုင်းမှ စတင်လေ့လာနိုင်ပါသည်။

ဒီသင်တန်းအတွက် ဘာသာစကား အထောက်အပံ့များ ရှိပါသည်။ ကျွန်တော်တို့၏ [ရနိုင်သော ဘာသာစကားများကို ဒီမှာ](../..) ကြည့်ပါ။

Generative AI မော်ဒယ်များဖြင့် ပထမဆုံး ဆောက်လုပ်မည်ဆိုပါက ကျွန်တော်တို့၏ [Generative AI For Beginners](https://aka.ms/genai-beginners) သင်တန်းကို ကြည့်ပါ။ အဆိုပါ သင်တန်းတွင် GenAI ဖြင့် ဆောက်လုပ်ခြင်းဆိုင်ရာ သင်ခန်းစာ 21 ချက် ပါဝင်သည်။

ဒီ repo ကို [ကြယ်ပေး (🌟) ရနိုင်ဖို့](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) မမေ့ပါနှင့်၊ အကုန်ကို လုပ်မယ်ဆိုရင် ဒီ repo ကို [fork လုပ်ပါ](https://github.com/microsoft/ai-agents-for-beginners/fork)။

### အခြား သင်ယူသူများနှင့် တွေ့ဆုံပြီး မေးခွန်းများကို ဖြေရှင်းလိုပါသလား

AI Agents ဆောက်လုပ်ရာတွင် အခက်အခဲသို့မဟုတ် မေးခွန်းများ ရလာပါက ကျွန်တော်တို့၏ Microsoft Foundry Discord တွင် သီးသန့် ဖိုရမ်အား ဝင်ပါ။ (https://aka.ms/ai-agents/discord)

### လိုအပ်သော အရာများ

ဒီသင်တန်း တစ်ခုချင်းစီတွင် code အား ဥပမာများပါဝင်ပြီး code_samples ဖိုလ်ဒါတွင် တွေ့နိုင်သည်။ သင့်ကိုယ်ပိုင် ကော်ပီတစ်ခု ဖန်တီးရန် [ဒီ repo ကို fork လုပ်ပါ](https://github.com/microsoft/ai-agents-for-beginners/fork)။

ဒီ လေ့ကျင့်ခန်းများရှိ code ဥပမာများတွင် Azure AI Foundry နှင့် GitHub Model Catalogs ကို Language Models နှင့် အဆက်အသွယ် 갖ရန် အသုံးပြုထားသည်။

- [Github Models](https://aka.ms/ai-agents-beginners/github-models) - အခမဲ့ / ကန့်သတ်
- [Azure AI Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure အကောင့် လိုအပ်သည်

ဒီသင်တန်းသည် Microsoft မှ ပေးသည့် အောက်ပါ AI Agent ဖရိမ်ဝက်များနှင့် ဝန်ဆောင်မှုများကိုလည်း အသုံးပြုသည် -

- [Microsoft Agent Framework (MAF) - New!](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Agent Service](https://aka.ms/ai-agents-beginners/ai-agent-service)
- [Semantic Kernel](https://aka.ms/ai-agents-beginners/semantic-kernel)
- [AutoGen](https://aka.ms/ai-agents/autogen)


ဒီသင်တန်းအတွက် code များကို ထုတ်လုပ်အသုံးပြုရန်နည်းလမ်းပိုမို လေ့လာလိုပါက [Course Setup](./00-course-setup/README.md) ကို သွားကြည့်ပါ။
