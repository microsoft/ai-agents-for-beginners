## 🌱 စတင်ရန်

ဒီသင်ခန်းစာတွင် AI အေးဂျင့်များ တည်ဆောက်ခြင်းအခြေခံကို ဖော်ပြသည့် သင်ခန်းစာ ၁၁ ခု ပါဝင်ပါတယ်။ သင်ခန်းစာတိုင်းမှာ မတူညီတဲ့ အကြောင်းအရာကို ဖော်ပြထားတာကြောင့် သင်စိတ်ဝင်စားရာနေရာက စတင်လေ့လာနိုင်ပါတယ်။

ဒီသင်ခန်းစာမှာ ဘာသာစကားအထောက်အပံ့လည်း ပါဝင်ပါတယ်။ [ဒီမှာ](../..) သွားပြီး သင့်လိုအပ်တဲ့ ဘာသာစကားကို ရွေးပါ။

Generative AI မော်ဒယ်များနှင့် ပထမဆုံးအကြိမ် လုပ်ဆောင်မယ်ဆိုရင် [Generative AI For Beginners](https://aka.ms/genai-beginners) သင်ခန်းစာကို ကြည့်ပါ။ ဒီသင်ခန်းစာမှာ GenAI နဲ့ ဆောက်လုပ်ခြင်းအကြောင်း သင်ခန်းစာ ၂၁ ခု ပါဝင်ပါတယ်။

[ဒီ repo ကို 🌟 (star)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) နှင့် [fork](https://github.com/microsoft/ai-agents-for-beginners/fork) လုပ်ပြီး ကိုယ်ပိုင် code ကို run လုပ်ဖို့ မမေ့ပါနဲ့။

### အခြားလေ့လာသူများနှင့် တွေ့ဆုံပြီး မေးခွန်းများကို ဖြေရှင်းပါ

AI အေးဂျင့်များ တည်ဆောက်ရာတွင် အခက်အခဲရှိပါက သို့မဟုတ် မေးခွန်းများရှိပါက [Azure AI Foundry Community Discord](https://aka.ms/ai-agents/discord) တွင် ကျွန်ုပ်တို့၏ အထူး Discord Channel ကို ဝင်ရောက်ပါ။

### သင်လိုအပ်တာများ

ဒီသင်ခန်းစာတွင် ပါဝင်တဲ့ code နမူနာများကို code_samples folder တွင် တွေ့နိုင်ပါတယ်။ [ဒီ repo ကို fork](https://github.com/microsoft/ai-agents-for-beginners/fork) လုပ်ပြီး ကိုယ်ပိုင် copy တစ်ခု ဖန်တီးပါ။

ဒီသင်ခန်းစာတွင် ပါဝင်တဲ့ code နမူနာများသည် Azure AI Foundry နှင့် GitHub Model Catalogs ကို အသုံးပြုထားပြီး Language Models များနှင့် အပြန်အလှန် ဆက်သွယ်ရန် အသုံးပြုထားပါတယ်-

- [Github Models](https://aka.ms/ai-agents-beginners/github-models) - အခမဲ့ / အကန့်အသတ်ရှိ
- [Azure AI Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure အကောင့်လိုအပ်

ဒီသင်ခန်းစာတွင် Microsoft မှ AI အေးဂျင့် frameworks နှင့် ဝန်ဆောင်မှုများကိုလည်း အသုံးပြုထားပါတယ်-

- [Azure AI Agent Service](https://aka.ms/ai-agents-beginners/ai-agent-service)  
- [Semantic Kernel](https://aka.ms/ai-agents-beginners/semantic-kernel)  
- [AutoGen](https://aka.ms/ai-agents/autogen)  

ဒီသင်ခန်းစာကို run လုပ်ရန် ပိုမိုသိရှိလိုပါက [Course Setup](./00-course-setup/README.md) ကို သွားကြည့်ပါ။

## ပံ့ပိုးမှု

ဤပရောဂျက်သည် ပံ့ပိုးမှုများနှင့် အကြံပြုချက်များကို ကြိုဆိုပါသည်။ အများစုသော ပံ့ပိုးမှုများအတွက် Contributor License Agreement (CLA) ကို သဘောတူရန် လိုအပ်ပါသည်။ သင့်ပံ့ပိုးမှုကို အသုံးပြုရန် ကျွန်ုပ်တို့အား အခွင့်အရေးပေးရန် သင့်တွင် အခွင့်အရေးရှိကြောင်း သက်သေပြရန် လိုအပ်ပါသည်။ အသေးစိတ်အချက်အလက်များအတွက် <https://cla.opensource.microsoft.com> ကို ကြည့်ပါ။

သင် pull request တင်သွင်းသောအခါ CLA bot သည် သင် CLA ပေးရန် လိုအပ်မလားကို အလိုအလျောက် ဆုံးဖြတ်ပြီး PR ကို သင့်တော်စွာ အဆင့်သတ်မှတ်ပေးပါမည် (ဥပမာ - status check, comment)။ bot မှ ပေးထားသော အညွှန်းများကို လိုက်နာပါ။ CLA ကို သင်တစ်ကြိမ်သာ လိုအပ်မည်ဖြစ်ပြီး CLA ကို အသုံးပြုသည့် repo အားလုံးတွင် သက်တမ်းရှိပါသည်။

ဤပရောဂျက်သည် [Microsoft Open Source Code of Conduct](https://opensource.microsoft.com/codeofconduct/) ကို လက်ခံထားပါသည်။
ပိုမိုသိရှိလိုပါက [Code of Conduct FAQ](https://opensource.microsoft.com/codeofconduct/faq/) ကို ကြည့်ရှုပါ သို့မဟုတ် [opencode@microsoft.com](mailto:opencode@microsoft.com) သို့ ဆက်သွယ်ပါ။

## အမှတ်တံဆိပ်များ

ဤပရောဂျက်တွင် ပရောဂျက်များ၊ ထုတ်ကုန်များ သို့မဟုတ် ဝန်ဆောင်မှုများအတွက် အမှတ်တံဆိပ်များ သို့မဟုတ် လိုဂိုများ ပါဝင်နိုင်ပါသည်။ Microsoft အမှတ်တံဆိပ်များ သို့မဟုတ် လိုဂိုများကို အသုံးပြုရန် ခွင့်ပြုချက်သည် [Microsoft's Trademark & Brand Guidelines](https://www.microsoft.com/legal/intellectualproperty/trademarks/usage/general) ကို လိုက်နာရမည်ဖြစ်သည်။
ဤပရောဂျက်၏ ပြင်ဆင်ထားသော ဗားရှင်းများတွင် Microsoft အမှတ်တံဆိပ်များ သို့မဟုတ် လိုဂိုများကို အသုံးပြုခြင်းသည် အလွဲသုံးစားမှု သို့မဟုတ် Microsoft အား ပံ့ပိုးမှုရှိကြောင်း အထင်မှားစေခြင်း မဖြစ်စေရန် လိုအပ်ပါသည်။
တတိယပါတီ၏ အမှတ်တံဆိပ်များ သို့မဟုတ် လိုဂိုများကို အသုံးပြုခြင်းသည် ထိုတတိယပါတီများ၏ မူဝါဒများကို လိုက်နာရမည်ဖြစ်သည်။
