## 🌱 စတင်ခြင်း

ဒီသင်တန်းမှာ AI ကုန်သည်တွေ တည်ဆောက်ခြင်းအခြေခံ အတန်းတွေ ပါဝင်ပါတယ်။ အတန်းတိုင်းမှာ မိမိတို့ဘာသာ ရေးသူရာစွဲ ထားတဲ့ ခေါင်းစဉ်တစ်ခုကို ဖော်ပြထားလို့ မည်သည့် အတန်းကနေစတင်လိုက်ပါ။

ဒီသင်တန်းအတွက် မျိုးဘာသာစကားများကို အထောက်အပံ့ ပေးထားသည်။ ကျွန်ုပ်တို့၏ [ရနိုင်သော ဘာသာစကားများကို ဒီနေရာတွင်](#-multi-language-support) ကြည့်ရှုပါ။

Generative AI မော်ဒယ်များနှင့် ပထမဆုံးအကြိမ် တည်ဆောက်ခြင်းဖြစ်ပါက၊ [Generative AI For Beginners](https://aka.ms/genai-beginners) သင်တန်းကို ကြည့်ပါ၊ ထိုသင်တန်းတွင် GenAI ဖြင့် တည်ဆောက်ခြင်းအတွက် အတန်း ၂၁ ခု ပါဝင်သည်။

ဒီ repo ကို [ကြယ်ပြု (🌟)](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ပြုလုပ်ရန် နှင့် [fork လုပ်ရန်](https://github.com/microsoft/ai-agents-for-beginners/fork) မမေ့ပါနှစ်ပါ။

### အခြားတတ်မြောက်နည်းသူများနှင့် တွေ့ဆုံ၍ မေးခွန်းများဖြေကြားပါ

AI ကုန်သည်များ တည်ဆောက်ရာတွင် ရပ်တန့်သွားပါက သို့မဟုတ် မေးခွန်းများ ရှိပါက Microsoft Foundry Discord အတွင်းရှိ [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) ၏ အထူးဖေါ်ပြထားသော Discord ချန်နယ်ထဲ ဝင်ပါ။

### လိုအပ်ချက်များ 

ဒီသင်တန်းရှိ အတန်းတိုင်းတွင် ကိုးဒ်နမူနာများ ပါဝင်ပြီး၊ အဆိုပါ ကိုးဒ်များကို code_samples ဖိုလ်ဒါထဲတွင် ရှာဖွေနိုင်သည်။ ကိုယ့်ရဲ့ မိတ္တူကို ဖန်တီးဖို့ [fork လုပ်ပါ](https://github.com/microsoft/ai-agents-for-beginners/fork)။

ဤ လေ့ကျင့်ခန်းများရှိ code နမူနာများတွင် Microsoft Agent Framework နှင့် Azure AI Foundry Agent Service V2 ကို အသုံးပြုသည်။

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure အကောင့် လိုအပ်သည်

ဒီ သင်တန်းတွင် Microsoft ကနေသုံးသော AI Agent framework များနှင့် ဝန်ဆောင်မှုများမှာ:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

တချို့ code နမူနာများတွင် OpenAI- ကိုက်ညီသော အခြား ပံ့ပိုးသူများဖြစ်သည့် [MiniMax](https://platform.minimaxi.com/) ကဲ့သို့ မော်ဒယ်များကိုလည်း ထောက်ခံပါတယ် (အပတ်ဒါး 204K token အထိ မော်ဒယ်များရှိသည်)။ ဖွဲ့စည်းမှုအသေးစိတ်များအတွက် [Course Setup](./00-course-setup/README.md) ကို ကြည့်ပါ။

ဒီ သင်တန်းအတွက် ကိုးဒ် run မည်ဆိုပါက [Course Setup](./00-course-setup/README.md) ထဲသို့ သွားကြည့်ပါ။

## အတွဲဝင်ခြင်း

ဤပရောဂျက်သည် အတွဲဝင်မှုများနှင့် အကြံပြုချက်များကို လက်ခံသည်။ များသောအားဖြင့် အတွဲဝင်ရန် Contributor License Agreement (CLA) ကို သဘောတူရန် လိုအပ်ပြီး သင်၏အတွဲဝင်မှုကို အသုံးပြုခွင့်ပေးသည်ဟု တင်ပြရပါမည်။ အသေးစိတ်အချက်အလက်များအတွက် <https://cla.opensource.microsoft.com> ကို ကြည့်ပါ။

သင်သည် pull request တင်သည့်အခါ CLA bot သည် သင့်အား CLA ပေးရန် လိုအပ်ကြောင်းကို အလိုအလျောက်သိပါလိမ့်မည်နှင့် PR ကို သင့်တော်စွာ အလှဆင်ပေးပါလိမ့်မည် (ဥပမာ- အခြေအနေစစ်ဆေးမှု၊ မှတ်ချက်)။ bot မှပေးသောညွှန်ကြားချက်များကိုသာ လိုက်နာပါ။ ကျွန်ုပ်တို့၏ CLA ကို အသုံးပြုနေသော repository များအတွင်း၌ တစ်ကြိမ်သာ လုပ်ဆောင်ရပါမည်။
