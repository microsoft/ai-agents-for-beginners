> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> To daje wszystko, czego potrzebujesz, aby ukończyć kurs przy znacznie szybszym pobieraniu.

## 🌱 Zaczynamy

Ten kurs zawiera lekcje omawiające podstawy tworzenia agentów AI. Każda lekcja obejmuje inny temat, więc zacznij, gdzie chcesz!

Kurs wspiera wiele języków. Zobacz [dostępne języki tutaj](../..). 

Jeśli to Twój pierwszy raz z modelami generatywnej AI, sprawdź nasz kurs [Generative AI dla początkujących](https://aka.ms/genai-beginners), który zawiera 21 lekcji dotyczących tworzenia z GenAI.

Nie zapomnij [oznaczyć gwiazdką (🌟) tego repozytorium](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) i [sforkować to repozytorium](https://github.com/microsoft/ai-agents-for-beginners/fork) aby uruchomić kod.

### Poznaj innych uczestników, uzyskaj odpowiedzi na swoje pytania

Jeśli utkniesz lub masz pytania dotyczące budowania agentów AI, dołącz do naszego dedykowanego kanału Discord w [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Czego potrzebujesz

Każda lekcja w tym kursie zawiera przykłady kodu, które znajdują się w folderze code_samples. Możesz [sforkować to repozytorium](https://github.com/microsoft/ai-agents-for-beginners/fork) aby utworzyć własną kopię.  

Przykłady kodu w tych ćwiczeniach wykorzystują Microsoft Agent Framework wraz z Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Wymagane konto Azure

Ten kurs korzysta z następujących frameworków i usług Agentów AI od Microsoftu:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


Więcej informacji o uruchamianiu kodu dla tego kursu znajdziesz w sekcji [Konfiguracja kursu](./00-course-setup/README.md).
