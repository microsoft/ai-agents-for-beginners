# бһҳбҹҒбһҡбҹҖбһ“бһ”бһҫбһҖбһ•бҹ’бһӣбһјбһңбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ AI Agents бһ“бһ·бһ„бһҖбһҡбһҺбһёбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢ Agent

бһҹбҹ’бһңбһ¶бһӮбһҳбһ“бҹҚбһҳбһҖбһҖбһ¶бһ“бҹӢбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶ **AI Agents бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһўбҹ’бһ“бһҖбһҗбҹ’бһҳбһё**! бһңбһӮбҹ’бһӮбһ“бҹҒбҹҮбһ•бҹ’бһҸбһӣбҹӢбһҮбһјбһ“бһўбҹ’бһ“бһҖбһ“бһјбһңбһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ вҖ” бһ“бһ·бһ„бһҖбһјбһҠбһҠбҹӮбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ–бһ·бһҸ вҖ” бһҠбһҫбһҳбҹ’бһ”бһёбһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһ”бһ„бҹ’бһҖбһҫбһҸ AI Agents бһ–бһёбһӮбһ“бҹ’бһӣбһ„бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бҹ”

бһҳбһҖбһҹбһҪбһҡбһҹбҹ’бһҠбһёбһ“бҹ…бһҖбҹ’бһ“бһ»бһ„ <a href="https://discord.gg/kzRShWzttr" target="_blank">бһҹбһ бһӮбһҳбһ“бҹҚ Azure AI Discord</a> вҖ” бһҳбһ¶бһ“бһўбҹ’бһ“бһҖбһҡбҹҖбһ“ бһ“бһ·бһ„бһўбҹ’бһ“бһҖбһ”бһ„бҹ’бһҖбһҫбһҸ AI бһҠбҹӮбһӣбһҡбһёбһҖбһҡбһ¶бһҷбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҹбҹҶбһҺбһҪбһҡбҹ”

бһҳбһ»бһ“бһ–бҹҒбһӣбһҷбһҫбһ„бһ…бһ¶бһ”бҹӢбһ•бҹ’бһҠбһҫбһҳбһҹбһ¶бһ„бһҹбһ„бҹӢбһҷбҹүбһ¶бһ„бһ–бҹҒбһүбһӣбҹҒбһү бһҒбҹ’бһүбһ»бҹҶбһ“бһ№бһ„бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҗбһ¶бһҸбһҫбһҷбһҫбһ„бһ бҹ…бһўбҹ’бһңбһёбһҗбһ¶ AI Agent бһ“бһ·бһ„бһ–бҹҒбһӣбһҺбһ¶бһҠбҹӮбһӣбһңбһ¶бһҡбһҹбһҮбһ¶бһҸбһ·бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһҖбҹ’бһ“бһ»бһ„бһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбҹ”

## бһҳбҹҒбһҡбҹҖбһ“бһ”бһүбҹ’бһҮбһ¶бһҖбҹӢ

бһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮбһӮбҹ’бһҡбһ”бһҠбһҺбҹ’бһҸбһ”бҹӢбҹ–

- бһҸбһҫ AI Agents бһҮбһ¶бһўбҹ’бһңбһё бһ“бһ·бһ„бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ•бҹ’бһҹбҹҒбһ„бҹ—бһҳбһ¶бһ“бһўбҹ’бһңбһёбһҒбҹ’бһӣбҹҮ
- бһҖбһ¶бһҡбһ„бһ¶бһҡбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢ AI Agents бһҠбҹӮбһӣбһҹбһҳбһҹбҹ’бһҡбһ”бһ”бҹҶбһ•бһ»бһҸ
- бһ§бһ”бһҖбһҡбһҺбҹҚбһўбһӮбҹ’бһӮбһ·бһҹбһ“бһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ“бһ№бһ„бһ”бҹ’бһҡбһҫбһ–бҹҒбһӣбһҡбһ…бһ“бһ¶бһңбһ·бһ’бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ Agentic

## бһӮбҹ„бһӣбһ”бҹҶбһҺбһ„бһҹбһ·бһҖбҹ’бһҹбһ¶

бһ“бҹ…бһ…бһ»бһ„бһ”бһүбҹ’бһ…бһ”бҹӢбһ“бҹғбһҳбҹҒбһҡбҹҖбһ“бһ“бҹҒбҹҮ бһўбҹ’бһ“бһҖбһӮбһҪбһҡбһҸбҹӮбһўбһ¶бһ…бҹ–

- бһҹбһјбһҳбһ–бһ“бҹ’бһҷбһӣбҹӢбһўбҹҶбһ–бһё AI Agent бһ“бһ·бһ„бһ—бһ¶бһ–бһҒбһ»бһҹбһӮбҹ’бһ“бһ¶бһҡбһңбһ¶бһ„бһңбһ¶бһўбҹ’бһ“бһҖбһҡбҹҖбһ“бһ“бһ№бһ„бһҠбҹҶбһҺбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ AI бһ‘бһјбһ‘бҹ…
- бһҠбһ№бһ„бһҗбһ¶бһҸбһҫбһ–бҹҒбһӣбһҺбһ¶бһӮбһҪбһҡбһ”бҹ’бһҡбһҫ AI Agent (бһ“бһ·бһ„бһ–бҹҒбһӣбһҺбһ¶бһ бһ¶бһҳбһғбһ¶бһҸбҹӢ)
- бһӮбһјбһҡбһ‘бһ¶бһ“бһҖбһ¶бһҡбһҡбһ…бһ“бһ¶бһңбһ·бһ’бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ Agentic бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһүбҹ’бһ бһ¶бһҖбҹ’бһ“бһ»бһ„бһҮбһёбһңбһ·бһҸбһ–бһ·бһҸ

## бһҖбһ¶бһҡбһ”бһүбҹ’бһҮбһ¶бһҖбҹӢбһўбҹҶбһ–бһё AI Agents бһ“бһ·бһ„бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ•бҹ’бһҹбҹҒбһ„бҹ—бһӮбҹ’бһ“бһ¶

### бһҸбһҫ AI Agents бһҮбһ¶бһўбҹ’бһңбһё?

бһ“бҹҒбҹҮбһҮбһ¶бһҡбһ”бҹҖбһ”бһҹбһ¶бһҳбһүбҹ’бһүбһ“бҹғбһҖбһ¶бһҡбһҷбһӣбҹӢбҹ–

> **AI Agents бһҮбһ¶бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҠбҹӮбһӣбһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһұбҹ’бһҷбһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ’бҹҶ (LLMs) бһҮбһ¶бһҖбҹӢбһҹбҹ’бһҠбҹӮбһ„ *бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһўбҹ’бһңбһёбһҳбһҪбһҷбһ”бһ¶бһ“* вҖ” бһҠбҹ„бһҷбһ•бҹ’бһҸбһӣбҹӢбһұбҹ’бһҷбһ–бһҪбһҖбһңбһ¶бһ§бһ”бһҖбһҡбһҺбҹҚ бһ“бһ·бһ„бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„бһҠбһҫбһҳбҹ’бһ”бһёбһўбһ“бһ»бһңбһҸбҹ’бһҸбһӣбһҫбһ–бһ·бһ—бһ–бһӣбҹ„бһҖ бһҳбһ·бһ“бһҳбҹӮбһ“бһ’бҹ’бһңбһҫбһҸбҹӮбһҖбһ¶бһҡбһҶбҹ’бһӣбһҫбһҷбһҸбһ”бһ“бһ№бһ„бһҹбҹҶбһҺбһҫбһҸбҹӮбһ”бҹүбһ»бһҺбҹ’бһҺбҹ„бҹҮбҹ”**

бһҷбһҫбһ„бһ‘бһ¶бһүбһ–бһ“бҹ’бһҷбһӣбҹӢбһ”бһ“бҹ’бһҸбһ·бһ…бҹ–

- **бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’** вҖ” AI Agent бһҳбһ·бһ“бһҳбҹӮбһ“бһҮбһ¶бһҡбһ”бһҹбҹӢбһҸбҹӮбһҳбһҪбһҷбһ‘бҹҒбҹ” бһңбһ¶бһҮбһ¶бһҖбһ¶бһҡбһ”бҹ’бһҡбһҳбһјбһӣбһ•бҹ’бһӮбһ»бҹҶбһӮбҹ’бһ“бһ¶бһҡбһ”бһҹбҹӢбһ•бҹ’бһ“бҹӮбһҖбһ•бҹ’бһҹбҹҒбһ„бҹ—бҹ” бһ“бҹ…бһ…бҹҶбһҺбһ»бһ…бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“ бһўбҹ’бһ“бһҖбһҡбһ¶бһӣбҹӢбһӮбҹ’бһ“бһ¶бһўбһ¶бһ…бһҡбһҖбһғбһҫбһүAgent бһ‘бһ¶бҹҶбһ„бһ”бһёбһ•бҹ’бһ“бҹӮбһҖбҹ–
  - **бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹ** вҖ” бһ‘бһёбһӣбҹҶбһ“бҹ…бһҠбҹӮбһӣAgent бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бҹ” бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҮбҹҶбһ“бһ¶бһ“бҹӢбһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ’бҹ’бһңбһҫбһҖбһ¶бһҡбһҖбһҖбҹӢбһҠбҹҶбһҺбһҫбһҡбһ‘бҹҒбһҹбһ…бһҡбһҺбҹҚ бһңбһ¶бһҮбһ¶бһңбҹҒбһ‘бһ·бһҖбһ¶бһҖбһҖбҹӢбһҠбҹҶбһ”бһјбһ„бһ•бҹ’бһ‘бһ¶бһӣбҹӢбҹ”
  - **бһ§бһ”бһҖбһҡбһҺбҹҚбһҜбһҖбһ‘бҹҒбһҹ** вҖ” бһңбһ·бһ’бһёбһҠбҹӮбһӣAgent бһўбһ¶бһ“бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бҹ” бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡвҖӢбһўбһ¶бһ…бһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһҳбһҫбһӣбһҖбһ¶бһҡбһ…бһјбһӣбһҡбһҪбһҳбһҹбһҺбҹ’бһӢбһ¶бһӮбһ¶бһҡ бһ¬бһҸбһҳбҹ’бһӣбҹғбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҷбһ“бҹ’бһҸбһ бҹ„бҹҮбҹ”
  - **бһ§бһ”бһҖбһҡбһҺбҹҚбһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡ** вҖ” бһңбһ·бһ’бһёбһҠбҹӮбһӣAgent бһ”бҹ’бһҡбһҸбһ·бһ”бһҸбҹ’бһҸбһ·бһҖбһ¶бһҡбһҺбҹҚбҹ” бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһўбһ¶бһ…бһ”бһүбҹ’бһҮбһ¶бһҖбһҖбҹӢбһ”бһ“бҹ’бһ‘бһ”бҹӢ бһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҖбһ¶бһҡбһ”бһүбҹ’бһ…бһ¶бһҖбҹӢ бһ¬бһӣбһ»бһ”бһ”бҹ„бҹҮбһ”бһ„бҹӢбһҖбһ¶бһҡбһҖбһҖбҹӢбҹ”

- **бһҳбҹүбһјбһҠбҹӮбһӣбһ—бһ¶бһҹбһ¶бһ’бҹҶ (Large Language Models)** вҖ” Agent бһҳбһ¶бһ“бһҳбһ»бһ“ LLMs бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮ LLMs бһ“бһ¶бҹҶбһІбҹ’бһҷAgent бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бһҖбһ¶бһ“бҹӢбһҸбҹӮбһҒбҹ’бһӣбһ¶бҹҶбһ„бҹ” бһ–бһҪбһҖбһңбһ¶бһўбһ¶бһ…бһҷбһӣбҹӢбһ—бһ¶бһҹбһ¶ бһ’бҹ’бһңбһҫбһҸбҹ’бһҡбһҪбһҸбһ–бһ·бһ“бһ·бһҸбҹ’бһҷбһ”бһҡбһ·бһ”бһ‘ бһ“бһ·бһ„бһ”бҹҶбһӣбҹӮбһ„бһҹбҹҶбһҺбһҫбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫбһҳбһ·бһ“бһ…бҹ’бһ”бһ¶бһҹбҹӢбһҮбһ¶бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһёбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһҖбҹҶбһҺбһҸбҹӢбҹ”

- **бһўбһ“бһ»бһңбһҸбҹ’бһҸбһ“бҹҚбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–** вҖ” бһ”бһҫбһӮбҹ’бһҳбһ¶бһ“бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’Agent, LLM бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһ”бһ„бҹ’бһҖбһҫбһҸбһўбһҸбҹ’бһҗбһ”бһ‘бҹ” бһ“бҹ…бһҖбҹ’бһ“бһ»бһ„бһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’Agent, LLM бһўбһ¶бһ… *бһўбһ“бһ»бһңбһҸбҹ’бһҸ* бһҮбҹҶбһ бһ¶бһ“ бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷвҖӢ, бһ бҹ… API, бһ•бҹ’бһүбһҫбһҹбһ¶бһҡбҹ”

- **бһ…бһјбһӣбһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһ§бһ”бһҖбһҡбһҺбҹҚ** вҖ” бһ§бһ”бһҖбһҡбһҺбҹҚбһҠбҹӮбһӣAgent бһўбһ¶бһ…бһ”бҹ’бһҡбһҫ бһ–бһ№бһ„бһ•бҹ’бһўбҹӮбһҖбһӣбһҫ (бҹЎ) бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбһҠбҹӮбһӣбһңбһ¶бһ’бҹ’бһңбһҫбһҖбһ¶бһҡ бһ“бһ·бһ„ (бҹў) бһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһўбһ—бһ·бһңбһҢбҹ’бһҚбһ”бһ¶бһ“бһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбҹ” бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһўбһ¶бһ…бһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҹбҹҶбһ”бһ»бһҸбҹ’бһҡбһҷбһ“бҹ’бһҸбһ бҹ„бҹҮ бһ”бҹүбһ»бһ“бҹ’бһҸбҹӮбһҳбһ·бһ“бһўбһ¶бһ…бһҖбҹӮбһ”бҹ’бһҡбҹӮбһўбһҸбһ·бһҗбһ·бһҮбһ“бһ‘бҹҒ вҖ” бһңбһ¶бһўбһ¶бһҹбҹ’бһҡбҹҗбһҷбһӣбһҫбһўбҹ’бһңбһёбһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ—бҹ’бһҮбһ¶бһ”бҹӢбҹ”

- **бһҳбһ“бҹ’бһҸбҹ’бһҡбһёбһңбһ·бһүбҹ’бһүбһ¶бһҺ + бһ…бҹҶбһҺбҹҒбҹҮбһҠбһ№бһ„** вҖ” Agent бһўбһ¶бһ…бһҳбһ¶бһ“бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбһҒбҹ’бһӣбһё (бһҹбһ“бҹ’бһ‘бһ“бһ¶бһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“) бһ“бһ·бһ„бһўбһ„бҹ’бһӮбһ…бһ„бһ…бһ¶бҹҶбһңбҹӮбһ„ (бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһўбһҸбһ·бһҗбһ·бһҮбһ“ бһ“бһ·бһ„бһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҹбһ„бҹ’бһҹбҹҗбһҷ)бҹ” бһ—бҹ’бһ“бһ¶бһҖбҹӢбһ„бһ¶бһҡбһ’бҹ’бһңбһҫбһҠбҹҶбһҺбһҫбһҡбһўбһ¶бһ… "бһ…бһ„бһ…бһ¶бҹҶ" бһўбҹ’бһ“бһҖбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸбһҖбҹ…бһўбһёбһҮбһ¶бһ”бҹӢбһ”бһ„бҹ’бһўбһҪбһ…бҹ”

### бһ”бҹ’бһҡбһ—бҹҒбһ‘бһ•бҹ’бһҹбҹҒбһ„бҹ—бһ“бҹғ AI Agents

бһҳбһ·бһ“бһҳбҹӮбһ“Agent бһ‘бһ¶бҹҶбһ„бһўбһҹбҹӢбһҸбҹ’бһҡбһјбһңбһ”бһ¶бһ“бһ”бһ„бҹ’бһҖбһҫбһҸбһҠбһјбһ…бһӮбҹ’бһ“бһ¶бһ‘бҹҒбҹ” бһҹбһјбһҳбһҳбһҫбһӣбһҸбһ¶бһҡбһ¶бһ„бһ…бҹӮбһҖбһ…бһ¶бһҷбһ”бҹ’бһҡбһ—бҹҒбһ‘бһҮбһ¶бһӣбҹҶбһҠбһ¶бһ”бҹӢбҹ–

## бһ–бҹҒбһӣбһҺбһ¶бһ…бһ¶бҹҶбһ”бһ¶бһ…бҹӢбһ”бҹ’бһҡбһҫ AI Agents

бһӮбҹ’бһҡбһ¶бһ“бҹӢбһҸбҹӮбһўбҹ’бһ“бһҖбһўбһ¶бһ…бһ”бҹ’бһҡбһҫAI Agent бһҳбһ·бһ“бһ‘бһ¶бһ“бҹӢбһҳбһ¶бһ“бһ“бҹҗбһҷбһҗбһ¶бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһ”бҹ’бһҡбһҫбһ“бҹ„бҹҮбһ‘бҹҒбҹ” бһ“бҹҒбҹҮбһҮбһ¶бһҹбҹ’бһҗбһ¶бһ“бһ—бһ¶бһ–бһҠбҹӮбһӣAgent бһҳбһ¶бһ“бһ—бһ¶бһ–бһӣбҹҒбһ…бһ’бҹ’бһӣбҹ„бҹ–

![When to use AI Agents?](../../../translated_images/km/when-to-use-ai-agents.54becb3bed74a479.webp)

- **бһ”бһүбҹ’бһ бһ¶бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ…бһ»бһ„бһҖбҹ’бһҡбҹ„бһҷ** вҖ” бһ–бҹҒбһӣбһҮбҹҶбһ бһ¶бһ“бһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷбһ”бһүбҹ’бһ бһ¶бһҳбһ·бһ“бһўбһ¶бһ…бһҖбҹҶбһҺбһҸбҹӢбһҮбһ¶бһҳбһ»бһ“бһ‘бҹҒбҹ” бһўбҹ’бһ“бһҖбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһІбҹ’бһҷLLM бһҡбһҖбһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҸбһ¶бһҳбһҹбҹҶбһҺбһҫбһҡбҹ”
- **бһҠбҹҶбһҺбһҫбһҡбһҖбһ¶бһҡбһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһ бһ¶бһ“** вҖ” бһҖбһ¶бһҡбһ„бһ¶бһҡбһҠбҹӮбһӣбһҸбҹ’бһҡбһјбһңбһҖбһ¶бһҡбһ”бҹ’бһҡбһҫбһ§бһ”бһҖбһҡбһҺбҹҚбһҹбҹ’бһ‘бһҫбһҡбһҹбһҡбһ»бһ”бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһ бһ¶бһ“ бһҳбһ·бһ“бһҳбҹӮбһ“бһҸбҹӮбһҹбҹ’бһңбҹӮбһ„бһҡбһҖбһҸбҹӮбһҳбһҪбһҷ бһ¬бһ”бһ„бҹ’бһҖбһҫбһҸбһҸбҹӮбһҳбһҪбһҷбҹ”
- **бһҖбһ¶бһҡбһҖбҹӮбһӣбһҳбҹ’бһўбһҸбһ¶бһҳбһ–бҹҒбһӣбһңбҹҒбһӣбһ¶** вҖ” бһ–бҹҒбһӣбһ…бһ„бҹӢбһІбҹ’бһҷбһ”бҹ’бһҡбһ–бҹҗбһ“бҹ’бһ’бһҮбһ¶бһ“бҹӢбһЎбһҫбһ„бһңбһ·бһүбһҠбҹ„бһҷбһ•бҹ’бһўбҹӮбһҖбһӣбһҫбһҳбһҸбһ·бһҷбҹ„бһ”бһӣбҹӢбһўбҹ’бһ“бһҖбһ”бҹ’бһҡбһҫ бһ¬бһҹбһүбҹ’бһүбһ¶бһ”бһҡбһ·бһҷбһ¶бһҖбһ¶бһҹбҹ”

бһҷбһҫбһ„бһ“бһ№бһ„бһҮбҹ’бһҡбһ¶бһ”бһ–бһ·бһҹбҹҒбһҹбһ–бҹҒбһӣбһҺбһ¶ (бһ“бһ·бһ„бһ–бҹҒбһӣбһҺбһ¶ *бһҳбһ·бһ“*) бһ”бҹ’бһҡбһҫ AI Agents бһҖбҹ’бһ“бһ»бһ„бһҳбҹҒбһҡбҹҖбһ“ **бһ”бһ„бҹ’бһҖбһҫбһҸбһ—бһ¶бһ–бһҮбһҝбһҮбһ¶бһҖбҹӢбһҖбҹ’бһ“бһ»бһ„ AI Agents** бһ“бҹ…бһ–бҹҒбһӣбһҖбҹ’бһҡбҹ„бһҷбһ“бҹғбһңбһӮбҹ’бһӮбһҹбһ·бһҖбҹ’бһҹбһ¶бҹ”

## бһҳбһјбһӣбһҠбҹ’бһӢбһ¶бһ“бһ“бҹғбһңбһ·бһ’бһёбһҠбҹ„бҹҮбһҹбҹ’бһҡбһ¶бһҷ Agentic

### бһҖбһ¶бһҡбһўбһ—бһ·бһңбһҢбҹ’бһҚ Agent

бһҡбһҝбһ„бһҠбҹҶбһ”бһјбһ„бһҠбҹӮбһӣбһўбҹ’бһ“бһҖбһ’бҹ’бһңбһҫбһ“бҹ…бһ–бҹҒбһӣбһ”бһ„бҹ’бһҖбһҫбһҸ Agent бһӮбһәбһҖбҹҶбһҺбһҸбҹӢ *бһўбҹ’бһңбһёбһҠбҹӮбһӣбһңбһ¶бһўбһ¶бһ…бһ’бҹ’бһңбһҫбһ”бһ¶бһ“* вҖ” бһ§бһ”бһҖбһҡбһҺбҹҚ бһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ– бһ“бһ·бһ„бһўбһ¶бһҖбһ”бҹ’бһ”бһҖбһ·бһҡбһ·бһҷбһ¶бҹ”

бһҖбҹ’бһ“бһ»бһ„бһңбһӮбҹ’бһӮбһ“бҹҒбҹҮ бһҷбһҫбһ„бһ”бҹ’бһҡбһҫ **бһҹбҹҒбһңбһ¶бһҖбһҳбҹ’бһҳ Azure AI Agent** бһҮбһ¶бһңбҹҒбһ‘бһ·бһҖбһ¶вҖӢбһҹбҹҶбһҒбһ¶бһ“бҹӢбҹ” бһңбһ¶бһӮбһ¶бҹҶбһ‘бҹ’бһҡбҹ–

- бһҳбҹүбһјбһҠбҹӮбһӣбһ–бһёбһўбҹ’бһ“бһҖбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһҠбһјбһ…бһҮбһ¶ OpenAI, Mistral бһ“бһ·бһ„ Meta (Llama)
- бһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһҳбһ¶бһ“бһ”бһүбҹ’бһүбһҸбҹ’бһҸбһ· бһ–бһёбһўбҹ’бһ“бһҖбһ•бҹ’бһӮбһҸбҹӢбһ•бҹ’бһӮбһ„бҹӢбһҠбһјбһ…бһҮбһ¶ Tripadvisor
- бһҖбҹҶбһҺбһҸбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚ OpenAPI 3.0 бһ‘бһҳбҹ’бһҡбһ„бҹӢбһҹбҹ’бһҸбһ„бҹӢбһҠбһ¶бһҡ

### бһӮбҹҶбһҡбһј Agentic

бһўбҹ’бһ“бһҖбһҹбһҳбҹ’бһҡбһ”бһҹбһҳбҹ’бһҡбһҪбһӣбһҮбһ¶бһҳбһҪбһҷ LLMs бһҸбһ¶бһҳбһҡбһҷбҹҲбһҹбҹҶбһҺбһҫбҹ” бһҮбһ¶бһҳбһҪбһҷAgent бһўбҹ’бһ“бһҖбһҳбһ·бһ“бһўбһ¶бһ…бһ”бһ„бҹ’бһҖбһҫбһҸбһҹбҹҶбһҺбһҫбһӮбҹ’бһҡбһ”бҹӢбһҷбҹүбһ¶бһ„бһҠбҹ„бһҷбһҠбҹғбһ‘бһ¶бһ“бҹӢбһҖбһ¶бһӣбһ”бһ…бҹ’бһ…бһ»бһ”бҹ’бһ”бһ“бҹ’бһ“бһ‘бҹҒ вҖ” Agent бһҸбҹ’бһҡбһјбһңбһ”бҹҶбһ–бҹҒбһүбһҹбһҖбһҳбҹ’бһҳбһ—бһ¶бһ–бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһҮбҹҶбһ бһ¶бһ“бҹ” бһ“бҹ„бҹҮбһ бһҫбһҷбһҮбһ¶бһҖбһ“бҹ’бһӣбҹӮбһ„бһҠбҹӮбһӣ **бһӮбҹҶбһҡбһј Agentic** бһ…бһјбһӣбһҡбһҪбһҳбҹ” бһңбһ¶бһҮбһ¶бһҷбһ»бһ‘бҹ’бһ’бһҹбһ¶бһҹбҹ’бһҸбҹ’бһҡбһҠбҹӮбһӣбһўбһ¶бһ…бһ”бҹ’бһҡбһҫбһЎбһҫбһ„бһңбһ·бһүбһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһҹбҹҶбһҺбһҫ бһ“бһ·бһ„бһ…бһҡбһ…бһ¶бһҡбһҮбһ¶бһҳбһҪбһҷ LLMs бһ“бҹ…бһӣбһҫбһңбһ·бһ’бһёбһ„бһ¶бһҷбһӣбҹ’бһҳбһҫбһҹ бһ“бһ·бһ„бһ‘бһ»бһҖбһ…бһ·бһҸбҹ’бһҸбһ”бһ¶бһ“бҹ”

бһңбһӮбҹ’бһӮбһ“бҹҒбҹҮбһ“бһ№бһ„бһ”бһ„бҹ’бһ бһ¶бһүбһҸбһ¶бһҳбһӮбҹҶбһҡбһј Agentic бһҠбҹӮбһӣбһ–бҹҒбһүбһ“бһ·бһҷбһҳбһ“бһ·бһ„бһҳбһ¶бһ“бһ”бҹ’бһҡбһҷбҹ„бһҮбһ“бҹҚбһ”бҹҶбһ•бһ»бһҸбҹ”

### бһҹбҹҠбһ»бһҳAgentic

бһҹбҹҠбһ»бһҳAgentic бһ•бҹ’бһҸбһӣбҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ”бҹ’бһӣбһ„бҹӢбһҳбһ»бһ“ бһҖбһҳбҹ’бһҳбһңбһ·бһ’бһё бһ“бһ·бһ„бһ бҹҒбһҠбҹ’бһӢбһ¶бһҡбһ…бһ“бһ¶бһҹбһҳбҹ’бһ–бҹҗбһ“бҹ’бһ’бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸ Agentбҹ” бһңбһ¶бһўбһ“бһ»бһүбҹ’бһүбһ¶бһҸбһІбҹ’бһҷбҹ–

- бһ—бҹ’бһҮбһ¶бһ”бҹӢбһ§бһ”бһҖбһҡбһҺбҹҚбһ“бһ·бһ„бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–
- бһҸбһ¶бһҳбһҠбһ¶бһ“бһўбҹ’бһңбһёбһҠбҹӮбһӣAgent бһҖбҹҶбһ–бһ»бһ„бһ’бҹ’бһңбһҫ (бһ“бһ·бһ„бһҖбҹӮбһ”бҹ’бһҡбҹӮбһ”бһүбҹ’бһ бһ¶)
- бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһҡбһҪбһҳбһӮбҹ’бһ“бһ¶бһ•бҹ’бһ‘бһ¶бҹҶбһ„Agent бһ…бҹ’бһҡбһҫбһ“

бһҖбҹ’бһ“бһ»бһ„бһңбһӮбҹ’бһӮбһ“бҹҒбҹҮ бһҷбһҫбһ„бһ•бҹ’бһҸбҹ„бһҸбһӣбһҫ **бһҹбҹҠбһ»бһҳAgent Microsoft (MAF)** бһҹбһҳбҹ’бһҡбһ¶бһ”бҹӢбһ”бһ„бҹ’бһҖбһҫбһҸAgent бһҮбһ¶бһ…бҹ’бһҡбһҫбһ“бһ”бҹ’бһҡбһҫбһ”бҹ’бһҡбһ¶бһҹбҹӢбһ”бһ¶бһ“бһҷбҹүбһ¶бһ„бһҳбһ¶бһ“бһҹбһҳбһҸбҹ’бһҗбһ—бһ¶бһ–бҹ”
