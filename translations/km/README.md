## 🌱 ការចាប់ផ្ដើម

មេរៀននេះមានមេរៀនគ្របកផ្នែកមូលដ្ឋាននៃការបង្កើតអ្នក代理 AI។ មេរៀននីមួយៗគ្របកពីប្រធានបទនៃខ្លួនវា ដូច្នេះសូមចាប់ផ្ដើមនៅកន្លែងដែលអ្នកចូលចិត្ត!

មានការគាំទ្រភាសាជាច្រើនសម្រាប់មេរៀននេះ។ ចុចទៅកាន់ [ភាសាដែលមាននៅទីនេះ](#-multi-language-support) ។

បើនេះជាលើកដំបូងរបស់អ្នកក្នុងការបង្កើតជាមួយម៉ូឌែល Generative AI សូមពិនិត្យមើលមេរៀន [Generative AI សម្រាប់អ្នកចាប់ផ្ដើម](https://aka.ms/genai-beginners) របស់យើង ដែលមានមេរៀន ២១ លើការបង្កើតជាមួយ GenAI ។

កុំភ្លေ្ចកបោះភ្លើងចិត្តច្បាស់នឹង [ផ្កាយ (🌟) នេះ repo](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) និង [បfork repo នេះ](https://github.com/microsoft/ai-agents-for-beginners/fork) ដើម្បីរត់កូដ។

### ជួបអ្នករៀនដទៃទៀត, ទទួលបានការឆ្លើយសំណួររបស់អ្នក

បើអ្នកមានបញ្ហា ឬមានសំណួរអំពីការបង្កើតអ្នក代理 AI សូមចូលរួមក្នុងជំនួយ Discord Channel ពិសេសរបស់យើងនៅ [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) ។

### អ្វីដែលអ្នកត្រូវការត្រូវមាន

មេរៀនគ្រប់មួយក្នុងមេរៀននេះមានឧទាហរណ៍កូដ ដែលអាចស្វែងរកបាននៅក្នុងថត code_samples។ អ្នកអាច [បfork repo នេះ](https://github.com/microsoft/ai-agents-for-beginners/fork) ដើម្បីបង្កើតច្បាប់ផ្ទាល់ខ្លួនរបស់អ្នក។

ឧទាហរណ៍កូដនៅក្នុងកម្មវិធីទាំងនេះប្រើ Microsoft Agent Framework ជាមួយ Azure AI Foundry Agent Service V2៖

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - តម្រូវឲ្យមានគណនី Azure

មេរៀននេះប្រើប្រាស់សំណុំអ្នក代理 AI និងសេវាកម្មខាងក្រោមពី Microsoft៖

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

ឧទាហរណ៍កូដខ្លះៗក៏គាំទ្រ​ផ្តល់ជូនអ្នកផ្គត់ផ្គង់ជំនួសដែលអាចប្រើបានជាមួយ OpenAI ដូចជា [MiniMax](https://platform.minimaxi.com/) ដែលផ្តល់ឲ្យមានម៉ូឌែលជារូបភាពនូវបរិបទធំ (ចំនួនដល់ម៉ាក 204K token)។ សូមមើល [ការតំឡើងមេរៀន](./00-course-setup/README.md) សម្រាប់ព័ត៌មានរចនាសម្ព័ន្ធ។

សម្រាប់ព័ត៌មានលម្អិតបន្ថែមអំពីការរត់កូដសម្រាប់មេរៀននេះ សូមចូលទៅកាន់ [ការតំឡើងមេរៀន](./00-course-setup/README.md) ។
