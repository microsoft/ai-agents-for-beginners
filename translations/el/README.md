## 🌱 Ξεκινώντας

Αυτό το μάθημα έχει μαθήματα που καλύπτουν τα θεμελιώδη της κατασκευής Πρακτόρων Τεχνητής Νοημοσύνης. Κάθε μάθημα καλύπτει το δικό του θέμα οπότε ξεκινήστε όποιο θέλετε!

Υπάρχει υποστήριξη πολλών γλωσσών για αυτό το μάθημα. Πηγαίνετε στις [διαθέσιμες γλώσσες εδώ](../..).

Αν είναι η πρώτη φορά που κατασκευάζετε με μοντέλα Γενετικής Τεχνητής Νοημοσύνης, δείτε το μάθημα μας [Γενετική Τεχνητή Νοημοσύνη για Αρχάριους](https://aka.ms/genai-beginners), το οποίο περιλαμβάνει 21 μαθήματα για την κατασκευή με GenAI.

Μην ξεχάσετε να [αποθηκεύσετε με αστέρι (🌟) αυτό το αποθετήριο](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) και να [κλωνοποιήσετε αυτό το αποθετήριο (fork)](https://github.com/microsoft/ai-agents-for-beginners/fork) για να τρέξετε τον κώδικα.

### Γνωρίστε άλλους μαθητές, λάβετε απαντήσεις στις ερωτήσεις σας

Αν κολλήσετε ή έχετε οποιεσδήποτε ερωτήσεις σχετικά με την κατασκευή Πρακτόρων Τεχνητής Νοημοσύνης, συμμετέχετε στο αποκλειστικό μας κανάλι Discord στο [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Τι χρειάζεστε

Κάθε μάθημα σε αυτό το μάθημα περιλαμβάνει παραδείγματα κώδικα, τα οποία μπορείτε να βρείτε στο φάκελο code_samples. Μπορείτε να [κλωνοποιήσετε αυτό το αποθετήριο](https://github.com/microsoft/ai-agents-for-beginners/fork) για να δημιουργήσετε το δικό σας αντίγραφο.

Τα παραδείγματα κώδικα σε αυτές τις ασκήσεις χρησιμοποιούν το Microsoft Agent Framework με το Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Απαιτείται Λογαριασμός Azure

Αυτό το μάθημα χρησιμοποιεί τα εξής πλαίσια και υπηρεσίες Πρακτόρων Τεχνητής Νοημοσύνης της Microsoft:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


Για περισσότερες πληροφορίες σχετικά με το πώς να τρέξετε τον κώδικα για αυτό το μάθημα, πηγαίνετε στο [Course Setup](./00-course-setup/README.md).
