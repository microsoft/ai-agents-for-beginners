## üå± Per iniziare

Questo corso contiene lezioni che coprono i fondamenti della creazione di agenti AI. Ogni lezione tratta un argomento specifico, quindi inizia da dove preferisci!

Questo corso dispone di supporto multilingue. Vai alle [lingue disponibili qui](../..). 

Se √® la prima volta che crei con modelli di AI generativa, dai un'occhiata al nostro corso [Generative AI For Beginners](https://aka.ms/genai-beginners), che include 21 lezioni sulla costruzione con GenAI.

Non dimenticare di [mettere una stella (üåü) a questo repo](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) e di [eseguire il fork di questo repo](https://github.com/microsoft/ai-agents-for-beginners/fork) per eseguire il codice.

### Incontra altri studenti, ottieni risposte alle tue domande

Se rimani bloccato o hai domande sulla creazione di agenti AI, unisciti al nostro canale Discord dedicato in [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Di cosa hai bisogno

Ogni lezione di questo corso include esempi di codice, che puoi trovare nella cartella code_samples. Puoi [eseguire il fork di questo repo](https://github.com/microsoft/ai-agents-for-beginners/fork) per creare una tua copia.  

Gli esempi di codice in questi esercizi utilizzano Azure AI Foundry e i GitHub Model Catalogs per interagire con i modelli di linguaggio:

- [Github Models](https://aka.ms/ai-agents-beginners/github-models) - Free / Limited
- [Azure AI Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Richiede account Azure

Questo corso utilizza inoltre i seguenti framework e servizi per agenti AI di Microsoft:

- [Microsoft Agent Framework (MAF) - New!](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Agent Service](https://aka.ms/ai-agents-beginners/ai-agent-service)
- [Semantic Kernel](https://aka.ms/ai-agents-beginners/semantic-kernel)
- [AutoGen](https://aka.ms/ai-agents/autogen)


Per maggiori informazioni sull'esecuzione del codice di questo corso, vai a [Course Setup](./00-course-setup/README.md).
