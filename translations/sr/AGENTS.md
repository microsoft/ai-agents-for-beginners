# AGENTS.md

## Преглед пројекта

Овај репозиторијум садржи "AI агенте за почетнике" - обиман образовни курс који учи све што је потребно за прављење AI агената. Курс се састоји од више од 15 лекција које покривају основе, шаблоне дизајна, оквире и продукциону имплементацију AI агената.

**Кључне технологије:**
- Python 3.12+
- Jupyter ноутбук за интерактивно учење
- AI оквири: Microsoft Agent Framework (MAF)
- Azure AI сервис: Microsoft Foundry, Azure AI Foundry Agent Service V2

**Архитектура:**
- Структура заснована на лекцијама (директоријуми 00-15+)
- Свака лекција садржи: README документацију, примере кода (Jupyter ноутбукове) и слике
- Подршка за више језика преко аутоматског система за превод
- Један Python ноутбук по лекцији који користи Microsoft Agent Framework

## Команде за подешавање

### Предуслови
- Python 3.12 или виша верзија
- Azure претплата (за Azure AI Foundry)
- Инсталиран и аутентификован Azure CLI (`az login`)

### Почетно подешавање

1. **Клонирајте или форкујте репозиторијум:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # ИЛИ
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Креирајте и активирајте Python виртуално окружење:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # На Виндоус-у: venv\Scripts\activate
   ```

3. **Инсталирајте зависности:**
   ```bash
   pip install -r requirements.txt
   ```

4. **Поставите променљиве окружења:**
   ```bash
   cp .env.example .env
   # Уредите .env са вашим API кључевима и крајњим тачкама
   ```

### Потребне променљиве окружења

За **Azure AI Foundry** (обавезно):
- `AZURE_AI_PROJECT_ENDPOINT` - крајња тачка Azure AI Foundry пројекта
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - име распореда модела (нпр. gpt-4o)

За **Azure AI Search** (лекција 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - крајња тачка Azure AI Search
- `AZURE_SEARCH_API_KEY` - API кључ за Azure AI Search

Аутентикација: Покрените `az login` пре покретања ноутбукова (користи `AzureCliCredential`).

Свака лекција садржи више Jupyter ноутбукова за различите оквире:

- Фајл шаблон: `*-python-agent-framework.ipynb`

## Упутства за Pull Request-ове

### Пре слања

1. **Тестирајте измене:**
   - Потпуно покрените погођене ноутбукове
   - Уверите се да све ћелије се извршавају без грешака
   - Проверите да ли су излази одговарајући

2. **Ажурирања документације:**
   - Ажурирајте README.md ако додајете нове концепте
   - Додајте коментаре у ноутбукове за комплекснији код
   - Осигурајте да markdown ћелије објашњавају сврху

3. **Измене фајлова:**
   - Избегавајте комитовање `.env` фајлова (користите `.env.example`)
   - Не комитујте `venv/` или `__pycache__/` директоријуме
   - Оставите излазе ноутбукова када илуструју концепте
   - Уклоните привремене и backup ноутбукове (`*-backup.ipynb`)

### Формат наслова PR-а

Користите описне наслове:
- `[Lesson-XX] Додај нови пример за <концепт>`
- `[Fix] Исправи типографску грешку у lesson-XX README`
- `[Update] Побољшај пример кода у lesson-XX`
- `[Docs] Ажурирај упутства за подешавање`

### Обавезне провере

- Ноутбукови треба да се извршавају без грешака
- README фајлови треба да буду јасни и прецизни
- Пратите постојеће обрасце кода у репозиторијуму
- Одржавајте доследност са осталим лекцијама

## Пројектно специфичан контекст

### Подршка за више језика

Овај репозиторијум користи аутоматски систем превода:
- Подржано 50+ језика
- Преводи се налазе у `/translations/<lang-code>/` директоријумима
- GitHub Actions workflow се брине о ажурирању превода
- Изворни фајлови су на енглеском у корену репозиторијума

### Структура лекције

Свaка лекција прати конзистентан образац:
1. Видео минијатура са линком
2. Писани садржај лекције (README.md)
3. Примери кода у више оквира
4. Циљеви учења и предуслови
5. Повезани додатни ресурси

### Именовање примера кода

Формат: `<lesson-number>-python-agent-framework.ipynb`
- `01-python-agent-framework.ipynb` - Лекција 1, MAF Python
- `14-sequential.ipynb` - Лекција 14, MAF напредни шаблони

### Специјални директоријуми

- `translated_images/` - Локализоване слике за преводе
- `images/` - Оригиналне слике за енглески садржај
- `.devcontainer/` - Конфигурација VS Code развојног контејнера
- `.github/` - GitHub Actions workflow и шаблони

### Зависности

Кључни пакети из `requirements.txt`:
- `agent-framework` - Microsoft Agent Framework
- `a2a-sdk` - Подршка за Agent-to-Agent протокол
- `azure-ai-inference`, `azure-ai-projects` - Azure AI сервис
- `azure-identity` - Azure аутентификација (AzureCliCredential)
- `azure-search-documents` - Интеграција Azure AI Search
- `mcp[cli]` - Подршка за Model Context Protocol
