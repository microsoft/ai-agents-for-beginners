>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> Ово вам даје све што вам је потребно да завршите курс уз много брже преузимање.

## 🌱 Почетак

Овај курс има лекције које покривају основе прављења AI агената. Свака лекција покрива своју тему, па почните где год желите!

Постоји подршка за више језика за овај курс. Погледајте [доступне језике овде](#-multi-language-support).

Ако први пут радите с генеративним AI моделима, погледајте наш курс [Generative AI For Beginners](https://aka.ms/genai-beginners), који укључује 21 лекцију о прављењу са GenAI.

Не заборавите да [заокружите (🌟) ово складиште](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) и [форкујете ово складиште](https://github.com/microsoft/ai-agents-for-beginners/fork) да бисте покренули код.

### Упознајте друге ученике, добијте одговоре на ваша питања

Ако запретите или имате било каквих питања о прављењу AI агената, придружите се нашем посебном Discord каналу у [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Шта вам је потребно

Свака лекција у овом курсу укључује примере кода који се налазе у фолдеру code_samples. Можете [форковати ово складиште](https://github.com/microsoft/ai-agents-for-beginners/fork) да бисте направили своју копију.

Примери кода у овим вежбама користе Microsoft Agent Framework са Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Потребан Azure налог

Овај курс користи следеће AI Agent оквире и услуге од Microsoft-а:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

Неки примери кода подржавају и алтернативне OpenAI компатибилне провајдере као што је [MiniMax](https://platform.minimaxi.com/), који нуди моделе са великим контекстом (до 204K токена). Погледајте [Course Setup](./00-course-setup/README.md) за детаље конфигурације.

За више информација о покретању кода за овај курс, идите на [Course Setup](./00-course-setup/README.md).
