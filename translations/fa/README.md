> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> این به شما همه چیز لازم برای تکمیل دوره را با سرعت دانلود بسیار بیشتر می‌دهد.

## 🌱 شروع کار

این دوره دروس مربوط به اصول ساخت عوامل هوش مصنوعی را پوشش می‌دهد. هر درس موضوع خودش را دارد، پس هر جایی که دوست دارید شروع کنید!

این دوره از پشتیبانی چندزبانه برخوردار است. به [زبان‌های موجود ما](#-multi-language-support) مراجعه کنید.

اگر برای اولین بار با مدل‌های هوش مصنوعی مولد کار می‌کنید، دوره [هوش مصنوعی مولد برای مبتدیان](https://aka.ms/genai-beginners) ما را ببینید، که شامل ۲۱ درس درباره ساخت با GenAI است.

فراموش نکنید که این مخزن را [ستاره (🌟) بدهید](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) و برای اجرای کد [از این مخزن فورک بگیرید](https://github.com/microsoft/ai-agents-for-beginners/fork).

### ملاقات با دیگر یادگیرندگان، دریافت پاسخ سوالات شما

اگر گیر کردید یا سوالی درباره ساخت عوامل هوش مصنوعی دارید، به کانال دیسکورد اختصاصی ما در [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) بپیوندید.

### آنچه نیاز دارید

هر درس در این دوره شامل مثال‌های کد است که در پوشه code_samples قرار دارند. می‌توانید [این مخزن را فورک کنید](https://github.com/microsoft/ai-agents-for-beginners/fork) تا نسخه خود را بسازید.

مثال‌های کد در این تمرین‌ها از Microsoft Agent Framework با سرویس Azure AI Foundry Agent V2 استفاده می‌کنند:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - نیاز به حساب Azure دارد

این دوره از فریم‌ورک‌ها و سرویس‌های زیر برای عامل‌های هوش مصنوعی از مایکروسافت استفاده می‌کند:

- [فریم‌ورک عامل مایکروسافت (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [سرویس Azure AI Foundry Agent نسخه ۲](https://aka.ms/ai-agents-beginners/ai-agent-service)

برخی نمونه‌های کد همچنین ارائه‌دهندگان سازگار با OpenAI alternative مانند [MiniMax](https://platform.minimaxi.com/) را پشتیبانی می‌کنند، که مدل‌های با زمینه بزرگ (تا ۲۰۴ هزار توکن) ارائه می‌دهد. برای جزئیات پیکربندی به [راه‌اندازی دوره](./00-course-setup/README.md) مراجعه کنید.

برای اطلاعات بیشتر درباره راه‌اندازی کد این دوره، به [راه‌اندازی دوره](./00-course-setup/README.md) بروید.

توافقنامه مجوز مشارکت‌کننده (CLA) دارد که اعلام می‌کند شما حق استفاده دارید و در واقع حقوق استفاده از مشارکتتان را به ما می‌دهید. برای جزئیات، بازدید کنید از <https://cla.opensource.microsoft.com>.

هنگامی که درخواست گرفتن (pull request) ارسال می‌کنید، ربات CLA به صورت خودکار تعیین می‌کند که آیا نیاز به ارائه
CLA دارید و درخواست را متناسب با آن علامت‌گذاری می‌کند (مثلاً بررسی وضعیت، نظر). فقط کافی است دستورالعمل‌های
ربات را دنبال کنید. شما فقط یک بار برای تمام مخازنی که از CLA ما استفاده می‌کنند باید این کار را انجام دهید.

این پروژه از [کد رفتاری متن‌باز مایکروسافت](https://opensource.microsoft.com/codeofconduct/) پیروی کرده است.
برای اطلاعات بیشتر به [پرسش‌های متداول کد رفتار](https://opensource.microsoft.com/codeofconduct/faq/) مراجعه کنید یا با [opencode@microsoft.com](mailto:opencode@microsoft.com) تماس بگیرید برای سوالات یا نظرات اضافی.
