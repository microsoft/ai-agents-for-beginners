## 🌱 시작하기

이 강의 과정은 AI 에이전트 구축의 기본을 다루는 수업들이 포함되어 있습니다. 각 수업은 고유의 주제를 다루므로 원하는 곳부터 시작하세요!

이 강의 과정은 다국어를 지원합니다. [여기에서 사용 가능한 언어](#-multi-language-support)를 확인하세요.

생성형 AI 모델로 처음 빌딩하는 경우, 21개의 강의로 구성된 [초보자를 위한 생성형 AI](https://aka.ms/genai-beginners) 강의 과정을 확인해보세요.

이 저장소에 [별(🌟)을 꼭 눌러주세요](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) 그리고 [포크해](https://github.com/microsoft/ai-agents-for-beginners/fork) 코드를 실행하세요.

### 다른 학습자 만나기, 질문 해결하기

AI 에이전트 구축에 막히거나 질문이 있을 때는 [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord)의 전용 Discord 채널에 참여하세요.

### 필요한 사항

이 과정의 각 수업에는 코드 예제가 포함되어 있으며, 코드는 code_samples 폴더에 있습니다. 자신만의 사본을 만들려면 [이 저장소를 포크하세요](https://github.com/microsoft/ai-agents-for-beginners/fork).

이 연습의 코드 예제는 Microsoft Agent Framework와 Azure AI Foundry Agent Service V2를 활용합니다:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure 계정 필요

이 과정에는 Microsoft의 다음 AI 에이전트 프레임워크 및 서비스가 사용됩니다:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

몇몇 코드 샘플은 대체 OpenAI 호환 공급자도 지원하는데, 예를 들어 대규모 컨텍스트 모델(최대 204K 토큰)을 제공하는 [MiniMax](https://platform.minimaxi.com/)가 있습니다. 구성 자세한 내용은 [Course Setup](./00-course-setup/README.md)을 확인하세요.

이 과정의 코드를 실행하는 방법에 대한 자세한 정보는 [Course Setup](./00-course-setup/README.md)에 있습니다.
