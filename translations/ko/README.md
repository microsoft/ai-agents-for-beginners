## 🌱 시작하기

이 강좌는 AI 에이전트 구축의 기본을 다루는 수업들로 구성되어 있습니다. 각 수업은 자체 주제를 다루므로 원하는 곳부터 시작하세요!

이 강좌에는 다국어 지원이 있습니다. [여기](#-다국어-지원)에서 지원 언어를 확인하세요.

한국어 번역 용어 일관성 기준은 [한국어 용어 기준서](./GLOSSARY_KO.md)를 참고하세요.

생성 AI 모델로 처음 작업하는 경우, 21개 수업으로 구성된 [초보자를 위한 생성 AI](https://aka.ms/genai-beginners) 강좌를 확인해 보세요.

코드를 실행하려면 [이 저장소](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst)를 별(🌟)표 표시하고 [포크](https://github.com/microsoft/ai-agents-for-beginners/fork)하는 것을 잊지 마세요.

### 다른 학습자 만나기, 질문하기

AI 에이전트 구축에 관한 문제가 있거나 질문이 있을 경우, [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord)의 전용 Discord 채널에 참여하세요.

### 필요한 것

이 강좌의 각 수업은 코드 예제를 포함하며, code_samples 폴더에서 찾을 수 있습니다. [저장소를 포크](https://github.com/microsoft/ai-agents-for-beginners/fork)하여 본인만의 복사본을 만들 수 있습니다.

이 연습 문제의 코드 예제는 Azure AI Foundry Agent Service V2와 함께 Microsoft Agent Framework를 활용합니다:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure 계정 필요

이 강좌에서는 Microsoft의 다음 AI 에이전트 프레임워크 및 서비스를 사용합니다:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

일부 코드 샘플은 [MiniMax](https://platform.minimaxi.com/)와 같은 OpenAI 호환 대체 공급자도 지원하며, 최대 204K 토큰까지 큰 문맥 모델을 제공합니다. 구성 세부 사항은 [강좌 설정](./00-course-setup/README.md)을 참조하세요.

이 강좌의 코드 실행에 대한 자세한 정보는 [강좌 설정](./00-course-setup/README.md)에서 확인하세요.
