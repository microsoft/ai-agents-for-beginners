# ஆரம்பக்காரர்களுக்கான AI முகவர்கள் - படிப்பு வழிகாட்டி மற்றும் பாடநெறி சுருக்கம்

இந்த வழிகாட்டி "அரம்பக்காரர்களுக்கான AI முகவர்கள்" பாடநெறியின் சுருக்கத்தை வழங்கி, AI முகவர்களை உருவாக்குவதற்கான முக்கிய கருத்துகள், கட்டமைப்புகள் மற்றும் வடிவமைப்பு மாதிரிகளை விளக்குகிறது.

## 1. AI முகவர்களுக்கு அறிமுகம்

**AI முகவர்கள் என்றால் என்ன?**
AI முகவர்கள் பெரும் மொழி மாதிரிகளின் (LLMs) திறன்களை **கருவிகள்**, **அறிவு**, மற்றும் **மemorி** இற்கான அணுகலை கொடுத்து விரிவாக்கும் அமைப்புகள் ஆகும். பயிற்சி தரவின் அடிப்படையில் மட்டுமே உரை உருவாக்கும் ஒரு சாதாரண LLM சந்தோஷகரனுக்குமிங்கள், ஒரு AI முகவர்:
- தனது சுற்றுப்புறத்தை **கண்டறியும்** (உள்ளீடுகள் அல்லது சென்சார்கள் மூலம்).
- ஒரு பிரச்சினையை தீர்க்க **தர்க்கம் செய்கிறது**.
- சுற்றுப்புறத்தை மாற்ற **செயல்** (செயல்படுத்திகள் அல்லது கருவி செயலாக்கம் மூலம்).

**ஒரு முகவரின் முக்கிய கூறுகள்:**
- **சுற்றுப்புறம்**: முகவர் நடவடிக்கையாற்றும் இடம் (எ.கா., ஒரு முன்பதிவு அமைப்பு).
- **சென்சார்கள்**: தகவல் சேகரிக்கும் முறைகள் (எ.கா., API வாசித்தல்).
- **செயல்படுத்திகள்**: நடவடிக்கைகளை மேற்கொள்ளும் முறைகள் (எ.கா., மின்னஞ்சல் அனுப்புதல்).
- **மூளை (LLM)**: எந்த நடவடிக்கைகளை எடுக்க வேண்டும் என்பதை திட்டமிட்டும் முடிவெடுக்கும் காரணக்கூரிய இயந்திரம்.

## 2. முகவரி கட்டமைப்புகள்

பாடநெறி முகவர்களை உருவாக்க **Microsoft Agent Framework (MAF)** மற்றும் **Azure AI Foundry Agent Service V2** ஐ பயன்படுத்துகிறது:

| கூறு | கவனம் | சிறந்த பயன் |
|-----------|-------|----------|
| **Microsoft Agent Framework** | முகவர்கள், கருவிகள் மற்றும் வேலைசெய்தல் சாத்தியமாக்கும் ஒருங்கிணைந்த Python/C# SDK | கருவிகள் கொண்ட முகவர்களை, அண்ணை முகவர்களின் வேலைசெய்தல் மற்றும் உற்பத்தி மாதிரிகள் கட்டமைக்க. |
| **Azure AI Foundry Agent Service** | நிர்வகிக்கபட்ட மேகம் ஓட்டுநிலை | கட்டுப்பாடுகளை உட்படுத்தி பாதுகாப்பான, தொலைவிற்கு தக்க, நிலை மேலாண்மை மற்றும் நம்பகத்தன்மை கொண்ட நிர்வகிப்பு. |

## 3. முகவரி வடிவமைப்பு மாதிரிகள்

வடிவமைப்பு மாதிரிகள் முகவர்கள் பிரச்சினைகளை நம்பகமாக தீர்க்க எவ்வாறு செயல்படுகிறார்கள் என்பதை அமைப்பதில் உதவுகின்றன.

### **கருவி பயன்பாடு மாதிரி** (பாடம் 4)
இந்த மாதிரி முகவர்களுக்கு வெளி உலகுடன் தொடர்பு கொள்ள உதவுகிறது.
- **கருத்து**: முகவரிக்கு "திட்டம்" வழங்கப்படுகிறது (கிடைக்கும் செயல்பாடுகள் மற்றும் அவற்றின் அளவுருக்கள் பட்டியல்). பயனர் கோரிக்கையின் அடிப்படையில் LLM *எந்த* கருவியை அழைக்க வேண்டும் மற்றும் *எந்த* அளவுருக்களுடன் அழைக்க வேண்டும் என்பதனை தீர்மானிக்கிறது.
- **வழிசெலுத்தல்**: பயனர் கோரிக்கை -> LLM -> **கருவி தேர்வு** -> **கருவி செயலாக்கம்** -> LLM (கருவி வெளியீடு உடன்) -> இறுதி பதில்.
- **பயன்பாடுகள்**: நேரடி தரவு மீட்டெடுப்பு (வானிலை, பங்கு விலை), கணக்கிடல், குறியீடு நிறைவேற்றல்.

### **திட்டமிடல் மாதிரி** (பாடம் 7)
இந்த மாதிரி பல படிகளாகும் கடினமான பணிகளை முகவர்கள் தீர்க்க உதவுகிறது.
- **கருத்து**: முகவர் ஒரு உயர்தர இலக்கை சிறிய துணைப் பணிகளின் தொடராகப் பிரிக்கிறது.
- **முறைகள்**:
  - **பணி பிரிப்பு**: "ஒரு பயணம் திட்டமிடு" என்பதன் கீழ் "விமானம் முன்பதிவு செய்", "ஹோட்டல் முன்பதிவு செய்", "கார் வாடகை எடு".
  - **மறுமொழி திட்டமிடல்**: முந்தைய படிகளின் முடிவுகளின் அடிப்படையில் திட்டத்தை மீண்டும் மதிப்பீடு செய்தல் (எ.கா., விமானம் நிறைவடைந்தால் வேறு தேதி தேர்வு செய்).
- **இணைப்பு**: அதிகமாக "திட்டமிடும்" முகவர் அமைத்து அதன் மூலம் கட்டமைக்கப்பட்ட (உதாரணமாக JSON) திட்டத்தை பிற முகவர்கள் செயல்படுத்துகிறார்கள்.

## 4. வடிவமைப்பு கோட்பாடுகள்

முகவர்களை வடிவமைக்கும்போது மூன்று பரிமாணங்களை கவனிக்கவும்:
- **இடம்**: முகவர்கள் மனிதர்களுடனும் அறிவுடனும் இணைக்க வேண்டும், அணுகக்கூடியதாகவும் தடுக்கப்படாத முறையிலும் இருக்க வேண்டும்.
- **நேரம்**: முகவர்கள் *முந்தைய* அனுபவம் இருந்து கற்றுக்கொள்ள, *தற்போது* தொடர்புடைய உதவிகளைக் கொடுக்க மற்றும் *எதிர்காலம்*விற்கு தானாக ஏற்படுத்திக் கொள்க.
- **மையம்**: உறுதியற்ற தன்மையை ஏற்றுக் கொண்டு, வெளிச்சம் மற்றும் பயனர் கட்டுப்பாடுகளின் மூலம் நம்பகத்தன்மையை உருவாக்கவும்.

## 5. முக்கிய பாடங்களின் சுருக்கம்

- **பாடம் 1**: முகவர்கள் மாதிரிகள் அல்ல, அமைப்புகள். அவை அறிந்துகொள்கின்றன, தர்க்கம் செய்கின்றன, செயல்படுகின்றன.
- **பாடம் 2**: Microsoft Agent Framework கருவி அழைப்பும் நிலை மேலாண்மையும் சிக்கல்களை ஒதுக்குமாறு செய்கிறது.
- **பாடம் 3**: வெளிச்சம் மற்றும் பயனர் கட்டுப்பாடுகளுடன் வடிவமைக்கவும்.
- **பாடம் 4**: கருவிகள் முகவரின் "கைகள்". LLM அதைப் பயன்படுத்த எப்படி என்று புரிந்து கொள்ள திட்ட வரையறை அவசியமாகும்.
- **பாடம் 7**: திட்டமிடல் முகவரின் "நிர்வாக செயல்பாடு", இது கடினமான வேலைசெய்தல்களை செய்ய உதவுகிறது.
