# AGENTS.md

## திட்ட அவலம்

இந்த கையேடு "துவக்கத்திற்கான AI முகவர்கள்" - AI முகவர்கள் கட்டுவதற்கான தேவையான அனைத்தையும் கற்றுக்கொள்ள ஒரு விரிவான கல்வி பாடத்தில் உள்ளது. இந்த பாடம் அடிப்படைகள், வடிவமைப்பு மாதிரிகள், கட்டமைப்புக்கள் மற்றும் AI முகவர்களின் உற்பத்தி இடத்தில் பயன்படுத்துதல் ஆகியவற்றை உள்ளடக்கிய 15+ பாடங்களை கொண்டுள்ளது.

**முக்கிய தொழில்நுட்பங்கள்:**
- Python 3.12+
- இடையாடல் கற்றலுக்கான Jupyter நோட்புக்கள்
- AI கட்டமைப்புகள்: Microsoft முகவர் கட்டமைப்பு (MAF)
- Azure AI சேவைகள்: Microsoft Foundry, Azure AI Foundry முகவர் சேவை V2

**கட்டமைப்பு:**
- பாடம் அடிப்படையிலான அமைப்பு (00-15+ கோப்புறைகள்)
- ஒவ்வொரு பாடத்திலும்: README ஆவணம், குறியீடு மாதிரிகள் (Jupyter நோட்புக்கள்), மற்றும் படங்கள் உள்ளன
- தானாக மொழி மாற்றம் செய்யும் அமைப்பின் மூலம் பன்மொழி ஆதரவு
- Microsoft முகவர் கட்டமைப்பை பயன்படுத்தும் ஒரு பைதான் நோட்புக் ஒவ்வொரு பாடத்திலும்

## அமைப்பு கட்டளைகள்

### முன் அம்சங்கள்
- Python 3.12 அல்லது அதற்கு மேலான பதிப்பு
- Azure சந்தா (Azure AI Foundry க்கான)
- Azure CLI நிறுவல் மற்றும் அங்கீகாரம் (`az login`)

### தொடக்க அமைப்பு

1. **கீழ்காணும் யாராச்சியை கிளோன் செய்யவும் அல்லது ஃபார்க் செய்யவும்:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # அல்லது
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Python மெய்நிகர் சூழல் உருவாக்கி செயலாக்கவும்:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # வின்டோஸ் இல்: venv\Scripts\activate
   ```

3. **தேவையான தொகுதிகள் நிறுவவும்:**
   ```bash
   pip install -r requirements.txt
   ```

4. **சூழல் மாறில்களை அமைக்கவும்:**
   ```bash
   cp .env.example .env
   # உங்கள் API விசைகளும் கொண்டடு அருகிய முகவரிகளும் கொண்ட .env ஐ தொகுக்கவும்
   ```

### தேவையான சூழல் மாறில்கள்

**Azure AI Foundry க்கான** (தேவையானவை):
- `AZURE_AI_PROJECT_ENDPOINT` - Azure AI Foundry திட்ட முடிவிடம்
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - மாதிரி இடைநிறுத்தல் பெயர் (எ.கா., gpt-4o)

**Azure AI தேடல்** (பாடம் 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - Azure AI தேடல் முடிவிடம்
- `AZURE_SEARCH_API_KEY` - Azure AI தேடல் API விசை

அங்கீகாரம்: நோட்புக்களை இயக்கும் முன் `az login` ஓட்டவும் (`AzureCliCredential` பயன்படுத்துகிறது).
