## 🌱 התחלה

קורס זה כולל שיעורים המכסים את היסודות של בניית סוכני בינה מלאכותית. כל שיעור עוסק בנושא משלו, אז התחילו מכל מקום שנראה לכם!

יש תמיכה רב-לשונית לקורס זה. עברו ל-[השפות הזמינות כאן](../..). 

אם זו הפעם הראשונה שאתם בונים עם מודלים של בינה גנרטיבית, צפו בקורס שלנו [בינה גנרטיבית למתחילים](https://aka.ms/genai-beginners), הכולל 21 שיעורים על בנייה עם GenAI.

אל תשכחו [לתת כוכב (🌟) למאגר זה](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ו[לעשות fork למאגר זה](https://github.com/microsoft/ai-agents-for-beginners/fork) כדי להריץ את הקוד.

### פגשו לומדים אחרים, קבלו תשובות לשאלותיכם

אם נתקעתם או יש לכם שאלות על בניית סוכני AI, הצטרפו לערוץ הדיסקורד הייעודי שלנו ב-[Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### מה צריך

כל שיעור בקורס זה כולל דוגמאות קוד, שנמצאות בתיקיית code_samples. אתם יכולים [לעשות fork למאגר זה](https://github.com/microsoft/ai-agents-for-beginners/fork) כדי ליצור עותק משלכם.  

דוגמאות הקוד בתרגילים אלו משתמשות ב-Microsoft Agent Framework עם Azure AI Foundry Agent Service V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - נדרש חשבון Azure

קורס זה משתמש במסגרת השירותים הבאים של Microsoft עבור סוכני AI:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


למידע נוסף על הרצת הקוד לקורס זה, עברו ל-[הגדרת הקורס](./00-course-setup/README.md).
