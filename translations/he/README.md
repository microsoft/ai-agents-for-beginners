> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> זה נותן לכם את כל מה שצריך כדי לסיים את הקורס עם הורדה מהירה הרבה יותר.

## 🌱 מתחילים

לקורס יש שיעורים שמכסים את היסודות של בניית סוכני בינה מלאכותית. כל שיעור מתמקד בנושא משלו, אז תתחילו מכל מקום שמתאים לכם!

יש תמיכה במגוון שפות לקורס זה. עיינו ברשימת השפות הזמינות [כאן](#-multi-language-support).

אם זו הפעם הראשונה שלכם בבניית דגמי AI יוצרים, בדקו את הקורס שלנו [בינה מלאכותית יוצרת למתחילים](https://aka.ms/genai-beginners), הכולל 21 שיעורים על בנייה עם GenAI.

אל תשכחו [לתת כוכב (🌟) למאגר זה](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ו-[לעשות Fork למאגר הזה](https://github.com/microsoft/ai-agents-for-beginners/fork) כדי להריץ את הקוד.

### הכירו לומדים אחרים, קבלו תשובות לשאלותיכם

אם אתם נתקעים או יש לכם שאלות לגבי בניית סוכני בינה מלאכותית, הצטרפו לערוץ הדיסקורד המיוחד שלנו ב-[Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### מה אתם צריכים

כל שיעור בקורס כולל דוגמאות קוד, שנמצאות בתיקיית code_samples. אתם יכולים [לעשות Fork למאגר הזה](https://github.com/microsoft/ai-agents-for-beginners/fork) כדי ליצור עותק משלכם.

דוגמאות הקוד בתרגילים אלה משתמשות ב-Microsoft Agent Framework עם שירות Azure AI Foundry Agent V2:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - דרוש חשבון Azure

הקורס משתמש במסגרת ועזרים הבאים לסוכני AI של מיקרוסופט:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [שירות Azure AI Foundry Agent V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

חלק מדוגמאות הקוד תומכות גם בספקים תואמי OpenAI אלטרנטיביים כגון [MiniMax](https://platform.minimaxi.com/), המציעים דגמי הקשר גדולים (עד 204K טוקנים). ראו את [הגדרות הקורס](./00-course-setup/README.md) לפרטים על התצורה.

למידע נוסף על הרצת הקוד של הקורס, עברו ל-[הגדרות הקורס](./00-course-setup/README.md).
