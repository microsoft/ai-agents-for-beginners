> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git

## 🌱 Alustamine

Selles kursuses on õppetunnid, mis käsitlevad AI-agentide ehitamise põhialuseid. Iga õppetund käsitleb oma teemat, nii et alusta kusagilt, mis sulle meeldib!

Selle kursuse jaoks on mitmekeelne tugi. Vaata meie [kättesaadavaid keeli siit](../..). 

Kui see on sinu esimene kord Generatiivsete tehisintellekti mudelitega ehitamisel, vaata meie [Generatiivne tehisintellekt algajatele](https://aka.ms/genai-beginners) kursust, mis sisaldab 21 õppetundi GenAI-ga ehitamisest.

Ära unusta sellele repole [tähte (🌟) andma](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) ja sellest [forkima](https://github.com/microsoft/ai-agents-for-beginners/fork) et koodi käivitada.

### Tutvu teiste õppijatega, saa oma küsimustele vastused

Kui jääd hätta või sul on küsimusi AI-agentide ehitamise kohta, liitu meie pühendatud Discord-kanaliga [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Mida vajad

Iga õppetunni juurde kuuluvad koodinäited, mis asuvad kaustas code_samples. Saad sellest [repost forkida](https://github.com/microsoft/ai-agents-for-beginners/fork), et luua oma koopia.  

Nendes harjutustes olevad koodinäited kasutavad Microsoft Agent Frameworki koos Azure AI Foundry Agent Service V2-ga:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Nõutav Azure konto

See kursus kasutab järgmisi Microsofti AI-agentide raamistikke ja teenuseid:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


Lisateabe saamiseks selle kursuse koodi käivitamise kohta, vaata [Kursuse seadistus](./00-course-setup/README.md).
