# AGENTS.md

## ప్రాజెక్ట్ అవలోకనం

ఈ సంకలనం "ఆరంభకులకు AI ఏజెంట్లు" అనే సమగ్ర విద్యా కోర్సును కలిగి ఉంది - AI ఏజెంట్లు నిర్మించడానికి అవసరమైన అన్ని విషయాలను నేర్పిస్తుంది. ఈ కోర్సు 15+ పాఠ్యాలుగా విభజించబడింది, ఇందులో ప్రాథమికాలు, డిజైన్ ప్యాటర్న్లు, ఫ్రేమ్‌వర్క్లు మరియు AI ఏజెంట్ల ఉత్పత్తి ఆవరణ గురించి ఉంది.

**ముఖ్య సాంకేతికతలు:**
- Python 3.12+
- ఇంటరాక్టివ్ అభ్యాసం కోసం Jupyter నోట్‌బుక్స్
- AI ఫ్రేమ్‌వర్క్లు: Microsoft Agent Framework (MAF)
- Azure AI సేవలు: Microsoft Foundry, Azure AI Foundry Agent Service V2

**కట్టడము:**
- పాఠ్యంపై ఆధారిత నిర్మాణం (00-15+ డైరెక్టరీలు)
- ప్రతి పాఠ్యానికి: README డాక్యుమెంటేషన్, కోడ్ సాంపిల్స్ (Jupyter నోట్‌బుక్స్), చిత్రాలు ఉంటాయి
- ఆటోమేటెడ్ అనువాద వ్యవస్థ ద్వారా బహుళభాషా మద్దతు
- Microsoft Agent Framework ఉపయోగించి ప్రతి పాఠ్యం కు ఒక Python నోట్‌బుక్

## సెటప్ కమాండ్లు

### అవసరమైన ప్రాథమికాలు
- Python 3.12 లేదా అంతకంటే పైగా
- Azure సబ్‌స్క్రిప్షన్ (Azure AI Foundry కోసం)
- Azure CLI సంస్థాపించబడినది మరియు ధృవీకరణ పొందినది (`az login`)

### ప్రారంభ సెటప్

1. **రిపాజిటరీను క్లోన్ లేదా ఫోర్‌క్ చేయండి:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # లేకపోతే
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Python వెర్చువల్ ఎన్‌విరాన్‌మెంట్ సృష్టించి యాక్టివేట్ చేయండి:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # విండోస్‌లో: venv\Scripts\activate
   ```

3. **ఆవశ్యక డిపెండెన్సీలను ఇన్‌స్టాల్ చేయండి:**
   ```bash
   pip install -r requirements.txt
   ```

4. **పర్యావరణ వేరియబుల్స్ సెట్ చేయండి:**
   ```bash
   cp .env.example .env
   # మీ API కీలు మరియు ఎండ్‌పాయింట్‌లతో .env ను సవరించండి
   ```

### అవసరమైన పర్యావరణ వేరియబుల్స్

**Azure AI Foundry** కోసం (ఆవశ్యకము):
- `AZURE_AI_PROJECT_ENDPOINT` - Azure AI Foundry ప్రాజెక్ట్ ఎండ్‌పాయింట్
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - మోడల్ డిప్లాయ్‌మెంట్ పేరు (ఉదా: gpt-4o)

**Azure AI Search** (పాఠ్యం 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - Azure AI Search ఎండ్‌పాయింట్
- `AZURE_SEARCH_API_KEY` - Azure AI Search API కీ

ప్రామాణికరణ: నోట్‌బుక్స్ ఆడించే ముందు `az login` నడపండి (`AzureCliCredential` ఉపయోగిస్తుంది).

## అభివృద్ధి వర్క్‌ఫ్లో

### Jupyter నోట్‌బుక్స్ నడపడం

ప్రతి పాఠ్యంలో వివిధ ఫ్రేమ్‌వర్క్ల కోసం బహుళ Jupyter నోట్‌బుక్స్ ఉంటాయి:

1. **Jupyter ప్రారంభించండి:**
   ```bash
   jupyter notebook
   ```

2. **ఒక పాఠ్య డైరెక్టరీకి వెళ్లండి** (ఉదా: `01-intro-to-ai-agents/code_samples/`)

3. **నోట్‌బుక్స్ తెరిచి నడపండి:**
   - `*-python-agent-framework.ipynb` - Microsoft Agent Framework (Python) ఉపయోగించి
   - `*-dotnet-agent-framework.ipynb` - Microsoft Agent Framework (.NET) ఉపయోగించి

### Microsoft Agent Framework తో పనిచేయడం

**Microsoft Agent Framework + Azure AI Foundry:**
- Azure సబ్‌స్క్రిప్షన్ అవసరం
- Agent Service V2 కోసం `AzureAIProjectAgentProvider` ఉపయోగిస్తుంది (Foundry పోర్టల్ లో ఏజెంట్లు కనిపిస్తాయి)
