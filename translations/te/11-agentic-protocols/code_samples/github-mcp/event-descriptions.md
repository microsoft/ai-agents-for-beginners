## ఈవెంట్ పేరు: LangGraph మరియు Azure Cosmos DBతో బహు-ఏజెంట్ API

## వివరణ

బహు-ఏజెంట్ AI అప్లికేషన్ల ఎదుగుదల తెలివైన వ్యవస్థలను రూపొందించుకునే విధానాన్ని మార్చేస్తోందా — అయితే వాటిని నిజ ప్రపంచ స్థాయిలో స్కేలబుల్ మరియు పనితీరు పరంగా ఎలా ఆర్కిటెక్ట్ చేయాలో తెలుసుకోవాలా? ఈ సెషన్‌లో, agent orchestration కోసం LangGraph, API కోసం FastAPI, మరియు state management, vector storage మరియు transactional data కోసం బ్యాక్‌బోన్‌గా Azure Cosmos DBని ఉపయోగించి నిర్మించిన ప్రొడక్షన్-గ్రేడ్ బహు-ఏజెంట్ అప్లికేషన్‌ను లోతుగా పరిశీలిస్తాం.

సవివరమైన కోడ్ వాక్‌త్రూ ద్వారా, మీరు retrieval-augmented generation (RAG), memory persistence, మరియు డైనామిక్ state ట్రాన్సిషన్లను సజావుగా సమన్వయపరిచే agent-driven వర్క్‌ఫ్లోని డిజైన్ చేసి అమలు చేయడం ఎలా అనేది చూడగలరు. మేము కవర్ చేయబోతున్న అంశాలు:

Agent collaboration with LangGraph for structured reasoning
Real-time chat history storage using Azure Cosmos DB - the same database that powers the chat history in ChatGPT, the fastest-growing AI agent application in history
Vector search for knowledge retrieval with Cosmos DB's native embeddings support
FastAPI’s async capabilities to keep interactions responsive and scalable
ఈ సెషన్ ముగిసే సమయానికి, ఆధునిక AI మరియు క్లౌడ్ ఇన్ఫ్రాస్ట్రక్చర్ శక్తిని వినియోగించి మీ స్వంత స్కేలబుల్, క్లౌడ్-నేటివ్ బహు-ఏజెంట్ అప్లికేషన్లను నిర్మించి డిప్లాయ్ చేయడానికి మీరు స్పష్టమైన బ్లూప్రింట్ పొందుతారు. మీరు AI ఇంజనీర్, క్లౌడ్ ఆర్కిటెక్ట్, లేదా Python డెవలపర్ ఏవైనా, ఈ టాక్ ప్రాయోగికమైన洞 సూచనలు మరియు అతి పరిశోధించబడిన నమూనాలు అందిస్తుంది తద్వారా తదుపరి తరం AI-శక్తిగల అప్లికేషన్లను నిర్మించగలుగుతారు

## వివరణ

ఈ సెషన్‌లో Microsoft Foundry ఉపయోగించి మీ Java అప్లికేషన్లలో అభివృద్ధి చెందిన AI ఏజెంట్ సామర్థ్యాలను ఎంబెడ్ చేయడం ఎలా అనేదాన్ని చూపిస్తాము, ప్రాజెక్ట్ లక్ష్యాలను సెట్ చేయడం, మోడల్స్‌తో ప్రయోగించడం మరియు సురక్షితంగా ప్రొడక్షన్-రెడీ పరిష్కారాలను స్కేల్ చేయడానికి డిప్లాయ్ చేయడం అవసరమైన దశలను కనిపెడతాము. మార్గంలో, GitHub Copilot (IntelliJ, VS Code, మరియు Eclipseలో) కోడింగ్ మరియు ప్రాంప్ట్ సృష్టిని ఎలా వేగవంతం చేయగలదో నేర్చుకుంటారు, అలాగే మోడల్ ఎంపిక, ఫైన్-ట్యూనింగ్, మరియు ఏజెంటిక్ వర్క్‌ఫ్లోలపై bästa పాటిత్వాలు బాధ్యతతో మరియు సమర్థవంతంగా అభివృద్ధి చేయడానికి సూచనలు అందజేస్తాయి. మీరు AI ఏజెంట్స్‌కు కొత్తవారు అయినా లేదా అధునాతన ఏజెంట్-నిర్మాణ సాంకేతికతల కోసం చూస్తున్నా, ఈ సెషన్ మీరు ఉన్న టూలింగ్‌తో తదుపరి స్థాయి అనుభవాలను డెలీవర్ చేయడానికి సన్నద్ధం చేస్తుంది.

## ఈవెంట్ పేరు: LangChain4j మరియు Dynamic Sessions ఉపయోగించి Java AI ఏజెంట్ల నిర్మాణం

## వివరణ

LangChain4jని Azure Container Apps (ACA) డైనమిక్ సెషన్‌లతో కలిసి Azure AI సేవలకు కనెక్ట్ చేసి మీ Java అప్లికేషన్లలో AI ఏజెంట్ల సామర్థ్యాన్ని అన్లాక్ చేయండి. ఈ సెషన్‌లో ఒక ప్రాక్టికల్ ఉదాహరణ చూపిస్తాము — రిమోట్ పరిమితితో ఇంటరాక్ట్ చేయగల ఏజెంట్, ఫైల్ మేనేజ్మెంట్ సహా. కస్టమ్ టూల్స్‌ను నిర్వచించి వాటిని ఏజెంట్ వర్క్‌ఫ్లోల్లో ఎలా అనుసంధానించాలో నేర్చుకోండి, మరియు Azure యొక్క స్కేలబుల్ ఇన్ఫ్రాస్ట్రక్చర్‌ను ఉపయోగించి తెలివైన, డైనమిక్ పరిష్కారాలను ఎలా డిప్లాయ్ చేయాలో తెలుసుకోండి.

## ఈవెంట్ పేరు: అనవసరమైన AI ఏజెంట్లు

## వివరణ

AI ఏజెంట్ల యొక్క సంభావ్య ప్రమాదాలను పరిశీలిస్తూ, సంభాషణాత్మక AI ప్లాట్‌‌ఫారమ్‌లలో నమ్మకమైన AI ప్రాక్టీసులను ఎంబెడ్ చేయడం ఎలా అనే సవాలును ఎదుర్కోవడానికి మాతో జత కండి! ఈ సెషన్‌లో అసమర్థ/అవినీతిపరమైన AI ఏజెంట్ల ఉదాహరణలలో లోతుగా విశ్లేషణ చేస్తాము — శకలభ్యమయ్యే మోడల్ వైఫల్యాల నమూనాలు, ప్రత్యర్థి జర్నలింగ్ (adversarial jailbreaks), మరియు నమ్మకాన్ని మరియు అనుగుణతను ధ్వంసం చేసే ఇతర ప్రమాదాలు చూపబడతాయి.

మేము Microsoft యొక్క నమ్మదగిన AI కోసం ఆధునిక టూల్స్‌ను వివరించి చూపిస్తాము, అందులో కంటెంటు ఫిల్టర్లు, రెడ్ టీమింగ్ వ్యూహాలు, మరియు అంచనాలు ఉంటాయి—లైవ్ డెమోలతో AI ఏజెంట్లు బాధ్యతతో మరియు అనారోగ్యంగా ఎలా ప్రవర్తిస్తాయో చూపిస్తాం.

🔥 మీరు ఈ సెషన్ నుంచి పొందే అంశాలు:
✅ ప్రయోజనాలను దోచుకోవడానికి ముందు AI ప్రమాదాలను ఎలా గుర్తించాలో మరియు తగ్గించాలో
✅ Azure AI Content Safetyని ఎలా డిప్లాయ్ చేయాలో మరియు ప్రమాదకర ప్రవర్తనను ఎలా గుర్తించాలో
✅ AI ఏజెంట్లను నమ్మదగ్గవిగా చేసుకోవడానికి రహస్య సూత్రాలు

హైప్, భారీ ప్రధానాంశాలు, మరియు మీ AIని చరిత్రలో సరైన బాగునుంచి ఉంచడానికి తప్పనిసరి వ్యూహాలతో నిండి ఉన్న సెషన్ కోసం సిద్ధంగా ఉండండి. సాదా AI మాత్రమే నిర్మించకండి—సాక్షాత్-maaina AIను నిర్మించండి!

## ఈవెంట్ పేరు: Azure AI Agent Service (.NET) తో మీ కోడ్-ఫస్ట్ యాప్‌ను నిర్మించండి

## వివరణ

Azure AI Agent Service సేవ మరియు SDK యొక్క సన్నని మిశ్రమం, ఇది బలమైన AI-ఆధారిత పరిష్కారాల అభివృద్ధిని సులభతరం చేస్తుంది. ఈ సెషన్‌లో, మీరు Azure మరియు C# ఉపయోగించి మీ స్వంత కోడ్-ఫస్ట్ AI ఏజెంట్‌ను ఎలా నిర్మించాలో నేర్చుకోవ خواهید — ఇది ప్రశ్నలకు సమాధానం చెప్పగలదు, డేటా విశ్లేషణ చేయగలదు, మరియు బాహ్య డేటా సోర్సులను ఏకీకృతం చేయగలదు. మీరు మరింత సంక్లిష్ట ఆర్కిటెక్చర్లు కూడా పరిశీలిస్తారు, అందులో బహు ఏజెంట్లు కలిసి పనిచేస్తుంటాయి.
