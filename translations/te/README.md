>
> ఈ రిపోజిటరీలో 50+ భాషా అనువాదాలు ఉన్నాయి, ఇది డౌన్‌లోడ్ పరిమాణాన్ని గణనీయంగా పెంచుతుంది. అనువాదాల 없이 క్లోన్ చేయడానికి, స్పార్స్ చెకౌట్ ఉపయోగించండి:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> ఇది మీకు కోర్సును పూర్తి చేయడానికి అవసరమైన అన్నిటినీ వేగంగా డౌన్లోడ్ ఇస్తుంది.

## 🌱 ప్రారంభించడం

ఈ కోర్సు AI ఏజెంట్లు నిర్మించే మూలాలను ఆవృతంగా తీసుకుంటుంది. ప్రతి పాఠం తన స్వంత విషయాన్ని కవర్ చేస్తుంది కాబట్టి మీరు ఇష్టమైన చోట నుంచి ప్రారంభించండి!

ఈ కోర్సుకు బహుభాషా మద్దతు ఉంది. మా [ఉపలబ్ధి భాషలు ఇక్కడ ఉన్నాయి](#-multi-language-support) చూడండి.

మీరు మొదటిసారిగా జనరేటివ్ AI మోడల్స్‌తో పని చేస్తునట్లయితే, మన [జనరేటివ్ AI ప్రారంభకులకు](https://aka.ms/genai-beginners) కోర్సును చూడండి, ఇది GenAI తో నిర్మించడంపై 21 పాఠాలను కలిగి ఉంది.

ఈ రిపోను [స్టార్ (🌟) చేయడం](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) మరవనండి మరియు [ఫోర్క్ చేయడం](https://github.com/microsoft/ai-agents-for-beginners/fork) కోడ్ నడపడానికి.

### ఇతర నేర్చుకునేవారిని కలవండి, మీ ప్రశ్నలకు సమాధానాలు పొందండి

మీరు ఏజెంట్లు నిర్మించడంలో ఇబ్బందిపడ్డ거나 ప్రశ్నలు ఉంటే, మన ప్రత్యేక Discord ఛానెల్‌లో చేరండి [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) లో.

### మీరు ఏమి అవసరం

ఈ కోర్సులో ప్రతి పాఠంలో కోడ్ ఉదాహరణలు ఉన్నాయి, అవి code_samples ఫోల్డర్‌లో గుంటగడ్చవచ్చు. మీరు [ఈ రిపోను ఫోర్క్](https://github.com/microsoft/ai-agents-for-beginners/fork) చేసుకుని మీ స్వంత ప్రతిని సృష్టించవచ్చు.

ఈ వ్యాయామ్‌లలో ఉన్న కోడ్ ఉదాహరణలు మైక్రోసాఫ్ట్ ఏజెంట్ ఫ్రేమ్‌వర్క్‌ను Azure AI Foundry ఏజెంట్ సర్వీస్ V2 తో ఉపయోగిస్తున్నాయి:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure ఖాతా అవసరం

ఈ కోర్సు Microsoft నుండి ఈ క్రింది AI ఏజెంట్ ఫ్రేమ్‌వర్క్‌లు మరియు సర్వీసులను ఉపయోగిస్తుంది:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

కొంత కోడ్ సాంప్ల్లు ప్రత్యామ్నాయ OpenAI-అనుకూల ప్రొవైడర్లు అయిన [MiniMax](https://platform.minimaxi.com/) వంటి పెద్ద కాన్‌టెక్స్ట్ మోడల్స్ (204K టోకెన్ల వరకు) ను మద్దతు ఇస్తాయి. కాన్ఫిగరేషన్ వివరాలకు [కోర్సు సెటప్](./00-course-setup/README.md) చూడండి.

ఈ కోర్సు కోడ్ నడిపేందుకు మరిన్ని వివరాల కోసం [కోర్సు సెటప్](./00-course-setup/README.md) చూడండి.

## సహకారం

ఈ ప్రాజెక్ట్ సహకారాలు మరియు సూచనలను స్వాగతిస్తుంది. బహుళ సహకారాలు మీరు ఒక
కాంట్రిబ్యూటర్ లైసెన్స్ ఒప్పందానికి (CLA) అంగీకరించాలని అవసరం, దీని ద్వారా మీరు మీ సహకారాన్ని ఉపయోగించుకునే హక్కు కలిగి ఉన్నారని మరియు నిజంగా అనుమతిస్తున్నారని ప్రకటిస్తారు. వివరాలకు సందర్శించండి <https://cla.opensource.microsoft.com>.

మీరు ఒక పుల్ రిక్వెస్ట్ సమర్పించినప్పుడు, CLA బాట్ ఆటోమేటిగ్గా మీరు CLA ఇవ్వాల్సిన అవసరాన్ని నిర్ధారిస్తుంది మరియు PR చక్కగా డెకరేట్ చేస్తుంది (ఉదా: స్థితి తనిఖీలు, వ్యాఖ్య). బాట్ సూచించిన దిశానిర్దేశాలను అనుసరించండి. మా CLA ఉపయోగించే అన్ని రిపోజ్‌లలో మీరు ఒకసారి మాత్రమే ఇది చేయాలి.

ఈ ప్రాజెక్ట్ [Microsoft Open Source Code of Conduct](https://opensource.microsoft.com/codeofconduct/) ని అనుసరిస్తుంది.
మరింత సమాచారం కోసం [Code of Conduct FAQ](https://opensource.microsoft.com/codeofconduct/faq/) చూడండి లేదా
ఏవైనా అదనపు ప్రశ్నలు లేదా వ్యాఖ్యల కోసం [opencode@microsoft.com](mailto:opencode@microsoft.com) ని సంప్రదించండి.
