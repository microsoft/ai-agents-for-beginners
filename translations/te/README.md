## 🌱 మొదలుకు

ఈ కోర్సులో AI ఏజెంట్లు నిర్మించడంలో ప్రాథమిక అంశాలు చర్చించబడ్డాయి. ప్రతి పాఠం దాని స్వంత అంశాన్ని కవర్ చేస్తుంది కాబట్టి మీరు ఇష్టపడిన చోట ప్రారంభించండి!

ఈ కోర్సుకు బహుభాషా మద్దతు ఉంది. మా [లభ్యమయ్యే భాషలను ఇక్కడ చూడండి](#-multi-language-support).

మీరు సృష్టిజన్య AI మోడల్స్‌తో మొదటిసారి పని చేస్తున్నట్లయితే, మా [Generative AI For Beginners](https://aka.ms/genai-beginners) కోర్సు చూడండి, ఇందులో GenAIతో నిర్మించేందుకు 21 పాఠాలు ఉన్నాయి.

దయచేసి ఈ రిపొని [స్టార్ (🌟) చేయడం](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) మరియు [ఫోర్క్ చేయడం](https://github.com/microsoft/ai-agents-for-beginners/fork) మర్చిపోకండి కోడ్ ని నడిపేందుకు.

### ఇతర అభ్యాసకులను కలవండి, మీ ప్రశ్నలకు జవాబులు పొందండి

మీరు ఏదైనా డౌటు లేదా AI ఏజెంట్లు నిర్మింప గురించి ప్రశ్నలు ఉంటే, మా ప్రత్యేక డిస్కార్డ్ చానెల్‌లో చేరండి [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### మీకు అవసరమయినవెప్పుడు

ఈ కోర్సు లో ప్రతి పాఠంలో కోడ్ ఉదాహరణలు ఉంటాయి, అవి code_samples ఫోల్డర్‌లో ఉంటాయి. మీరు [ఈ రిపోను ఫోర్క్](https://github.com/microsoft/ai-agents-for-beginners/fork) చేసుకోవచ్చు మీ స్వంత కాపీకి.

ఈ వ్యాయామాలలోని కోడ్ ఉదాహరణలు Microsoft Agent Frameworkని Azure AI Foundry Agent సేవ V2తో ఉపయోగిస్తాయి:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure ఖాతా అవసరం

ఈ కోర్సు Microsoft నుండి క్రింది AI ఏజెంట్ ఫ్రేమ్‌వర్క్లు మరియు సేవలను ఉపయోగిస్తుంది:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

కొన్ని కోడ్ సాంపిల్స్ ప్రత్యామ్నాయ OpenAI-అనుగుణమైన ప్రొవైడర్‌లను మద్దతు ఇస్తాయి, ఉదా. [MiniMax](https://platform.minimaxi.com/), ఇది భారీ-సందర్భ మోడల్స్ అందిస్తుంది (204K టోకెన్ల వరకు). సెట్ అప్ వివరాలకు [Course Setup](./00-course-setup/README.md) చూడండి.

ఈ కోర్సు కోడ్ నడిపేందుకు మరిన్ని వివరాలకు [Course Setup](./00-course-setup/README.md) వద్దికి సందర్శించండి.
