यो रिपोजिटरीमा "सुरुवातीहरूका लागि AI एजेन्टहरू" समावेश छन् - AI एजेन्टहरू बनाउन आवश्यक सबै कुरा सिकाउने एक व्यापक शैक्षिक कोर्स । कोर्समा १५+ पाठहरू छन् जसले आधारभूत ज्ञान, डिजाइन ढाँचाहरू, फ्रेमवर्कहरू, र AI एजेन्टहरूको उत्पादन डिप्लोइमेन्टमा समेट्छ ।

**प्रमुख प्रविधिहरू:**
- Python 3.12+
- अन्तरक्रियात्मक सिकाइको लागि Jupyter नोटबुकहरू
- AI फ्रेमवर्कहरू: Microsoft Agent Framework (MAF)
- Azure AI सेवाहरू: Microsoft Foundry, Azure AI Foundry Agent Service V2

**आर्किटेक्चर:**
- पाठ आधारित संरचना (००-१५+ डिरेक्टरीहरू)
- प्रत्येक पाठमा समावेश हुन्छ: README दस्तावेजीकरण, कोड नमूनाहरू (Jupyter नोटबुकहरू), र तस्बिरहरू
- स्वचालित अनुवाद प्रणाली मार्फत बहुभाषिक समर्थन
- प्रत्येक पाठमा Microsoft Agent Framework प्रयोग गरी एक Python नोटबुक

## सेटअप कमाण्डहरू

### पूर्वआवश्यकताहरू
- Python 3.12 वा माथि
- Azure सदस्यता (Azure AI Foundry का लागि)
- Azure CLI स्थापना र प्रमाणीकरण (`az login`)

### प्रारम्भिक सेटअप

1. **रिपोजिटरी क्लोन वा फोर्क गर्नुहोस्:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # वा
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Python भर्चुअल वातावरण सिर्जना र सक्रिय गर्नुहोस्:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # विन्डोजमा: venv\Scripts\activate
   ```

3. **निर्भरता स्थापना गर्नुहोस्:**
   ```bash
   pip install -r requirements.txt
   ```

4. **पर्यावरण चरहरू सेट गर्नुहोस्:**
   ```bash
   cp .env.example .env
   # तपाईंको API कुञ्जीहरू र अन्तबिन्दुहरूसहित .env सम्पादन गर्नुहोस्
   ```

### आवश्यक पर्यावरण चरहरू

**Azure AI Foundry** का लागि (आवश्यक):
- `AZURE_AI_PROJECT_ENDPOINT` - Azure AI Foundry परियोजना अन्त बिन्दु
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - मोडेल डिप्लोइमेन्ट नाम (जस्तै, gpt-4o)

**Azure AI Search** (पाठ ०५ - RAG) को लागि:
- `AZURE_SEARCH_SERVICE_ENDPOINT` - Azure AI Search अन्त बिन्दु
- `AZURE_SEARCH_API_KEY` - Azure AI Search API कुञ्जी

प्रमाणीकरण: नोटबुकहरू चलाउनु अघि `az login` चलाउनुहोस् (यसले `AzureCliCredential` प्रयोग गर्छ)।
