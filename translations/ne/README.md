## 🌱 सुरु गर्ने तरिका

यस कोर्समा AI एजेन्ट निर्माणका आधारभूत सिद्धान्तहरू समेटिएका पाठहरू छन्। प्रत्येक पाठले आफ्नो विषय समेट्छ, त्यसैले तपाईं जहाँ मन लाग्यो त्यहाँबाट सुरु गर्नुहोस्!

यस कोर्सका लागि बहुभाषिक समर्थन छ। हाम्रो [उपलब्ध भाषाहरू यहाँ](#-multi-language-support) हेर्नुहोस्।

यदि तपाईं पहिलो पटक जनरेटिभ AI मोडेलहरूसँग निर्माण गर्दै हुनुहुन्छ भने, हाम्रो [जनरेटिभ AI प्रारम्भकर्ताका लागि](https://aka.ms/genai-beginners) कोर्स हेर्नुहोस्, जसमा GenAI सँग निर्माण गर्ने २१ पाठहरू समावेश छन्।

यस रिपोमा [तारा (🌟) दिन नबिर्सनुहोस्](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) र कोड चलाउनका लागि यसलाई [fork गर्नुहोस्](https://github.com/microsoft/ai-agents-for-beginners/fork)।

### अन्य सिक्नेलाई भेट्नुहोस्, तपाईंका प्रश्नहरूको उत्तर पाउनुहोस्

यदि तपाईं अड्किनुभयो वा AI एजेन्टहरू विकास गर्ने बारे कुनै प्रश्न छ भने, हाम्रो विशेष Discord च्यानल [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) मा सामेल हुनुहोस्।

### तपाईंलाई के चाहिन्छ

यस कोर्सका प्रत्येक पाठमा कोड उदाहरणहरू छन्, जुन code_samples फोल्डरमा भेट्न सक्नुहुन्छ। तपाईं आफ्नो प्रतिलिपि बनाउनको लागि [यस रिपोलाई fork गर्न सक्नुहुन्छ](https://github.com/microsoft/ai-agents-for-beginners/fork)।

यी अभ्यासहरूमा कोड उदाहरणहरूले Microsoft Agent Framework र Azure AI Foundry Agent Service V2 प्रयोग गर्छन्:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure खाता आवश्यक छ

यस कोर्सले Microsoft का निम्न AI Agent फ्रेमवर्क र सेवाहरू प्रयोग गर्छ:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

केही कोड नमूनाहरूले वैकल्पिक OpenAI-संग मेल खाने प्रदायकहरू जस्तै [MiniMax](https://platform.minimaxi.com/) समर्थन गर्छन्, जुन ठूलो सन्दर्भ मोडेलहरू (अधिकतम २०४K टोकनसम्म) प्रस्ताव गर्दछ। कन्फिगरेसन विवरणका लागि [कोर्स सेटअप](./00-course-setup/README.md) हेर्नुहोस्।

यस कोर्सको लागि कोड चलाउने थप जानकारीका लागि [कोर्स सेटअप](./00-course-setup/README.md) जानुहोस्।
