## 🌱 सुरु गर्दै

यस कोर्समा AI एजेन्टहरू निर्माण गर्ने आधारभूत कुरा सिक्ने पाठहरू छन्। प्रत्येक पाठले आफ्नै विषयलाई समेट्छ त्यसैले तपाईं जहाँ चाहनुहुन्छ त्यहाँबाट सुरु गर्न सक्नुहुन्छ!

यस कोर्समा बहुभाषी समर्थन छ। हाम्रो [उपलब्ध भाषाहरू यहाँ जानुहोस्](#-multi-language-support)।

यदि तपाईं पहिलो पटक Generative AI मोडेलहरूसँग निर्माण गर्दै हुनुहुन्छ भने, हाम्रो [Generative AI For Beginners](https://aka.ms/genai-beginners) कोर्स हेर्नुहोस्, जसमा GenAI सँग निर्माण गर्ने २१ पाठहरू समावेश छन्।

यो रिपोजिटरीलाई [स्टार (🌟) दिन नबिर्सनुहोस्](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) र [fork गर्न](https://github.com/microsoft/ai-agents-for-beginners/fork) कोड चलाउन।

### तपाईंलाई के चाहिन्छ

यस कोर्सका प्रत्येक पाठमा कोड उदाहरणहरू छन्, जुन code_samples फोल्डरमा फेला पार्न सकिन्छ। तपाईं आफ्नो प्रतिलिपि बनाउन [यो रिपो फोर्क गर्न](https://github.com/microsoft/ai-agents-for-beginners/fork) सक्नुहुन्छ।

यी अभ्यासहरूमा कोड उदाहरणहरूले Microsoft Agent Framework सँग Azure AI Foundry Agent Service V2 प्रयोग गर्दछन्:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure खाता आवश्यक

यस कोर्सले Microsoft का निम्न AI Agent फ्रेमवर्कहरू र सेवाहरू प्रयोग गर्दछ:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

कतिपय कोड नमूनाहरू वैकल्पिक OpenAI-संगत प्रदायकहरू जस्तै [MiniMax](https://platform.minimaxi.com/) पनि समर्थन गर्दछ, जसले ठूलो सन्दर्भ मोडेलहरू (२०४K टोकन्स सम्म) उपलब्ध गराउँछ। कन्फिगरेसन विवरणहरूका लागि [कोर्स सेटअप](./00-course-setup/README.md) हेर्नुहोस्।

यस कोर्सका कोड चलाउने बारे थप जानकारीका लागि [कोर्स सेटअप](./00-course-setup/README.md) हेर्नुहोस्।
