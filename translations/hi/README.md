>
> इस रिपॉजिटरी में 50+ भाषाओं में अनुवाद शामिल हैं जो डाउनलोड आकार को काफी बढ़ा देता है। बिना अनुवाद के क्लोन करने के लिए, sparse checkout का उपयोग करें:
>
> **Bash / macOS / Linux:**
> ```bash
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone '/*' '!translations' '!translated_images'
> ```
>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git
> cd ai-agents-for-beginners
> git sparse-checkout set --no-cone "/*" "!translations" "!translated_images"
> ```
>
> यह आपको तेज़ डाउनलोड के साथ कोर्स पूरा करने के लिए सब कुछ प्रदान करता है।

## 🌱 शुरुआत कैसे करें

यह कोर्स AI एजेंट्स बनाने की बुनियादी बातों को कवर करता है। हर पाठ अपने विषय को कवर करता है इसलिए आप जहाँ चाहें वहां से शुरू कर सकते हैं!

इस कोर्स के लिए बहुभाषी समर्थन उपलब्ध है। हमारे [यहाँ उपलब्ध भाषाओं](../..) को देखें। 

यदि यह आपका पहली बार जनरेटिव AI मॉडल के साथ निर्माण कर रहे हैं, तो हमारा [Generative AI For Beginners](https://aka.ms/genai-beginners) कोर्स देखें, जिसमें GenAI के साथ निर्माण पर 21 पाठ शामिल हैं।

इस रिपॉजिटरी को [स्टार (🌟) करना](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) और कोड चलाने के लिए [फोर्क करना](https://github.com/microsoft/ai-agents-for-beginners/fork) मत भूलें।

### अन्य शिक्षार्थियों से मिलें, अपने सवालों के जवाब पाएं

यदि आप अटक जाएं या AI एजेंट्स के निर्माण के बारे में कोई सवाल हो, तो हमारे समर्पित Discord चैनल में जुड़ें [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) में।

### आपको क्या चाहिए

इस कोर्स के प्रत्येक पाठ में कोड उदाहरण शामिल हैं, जिन्हें code_samples फ़ोल्डर में पाया जा सकता है। आप अपनी कॉपी बनाने के लिए [इस रिपॉजिटरी को फोर्क](https://github.com/microsoft/ai-agents-for-beginners/fork) कर सकते हैं।  

इन अभ्यासों में कोड उदाहरण Microsoft Agent Framework के साथ Azure AI Foundry Agent Service V2 का उपयोग करते हैं:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure खाता आवश्यक

यह कोर्स Microsoft से निम्न AI एजेंट फ्रेमवर्क और सेवाओं का उपयोग करता है:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)


इस कोर्स के लिए कोड चलाने के बारे में अधिक जानकारी के लिए, [Course Setup](./00-course-setup/README.md) देखें।

Contributor License Agreement (CLA) पर सहमत होना आवश्यक है, जिसमें आप घोषणा करते हैं कि आपके पास
अपने योगदान का उपयोग करने का अधिकार है और आप इसे प्रदान करते हैं। विवरण के लिए देखें <https://cla.opensource.microsoft.com>।

और PR को उपयुक्त रूप से सजाएगा (जैसे, स्थिति जांच, टिप्पणी)। बस बॉट के निर्देशों का पालन करें।
आपको यह प्रक्रिया हमारे सभी CLA उपयोग करने वाले रिपोज़िटरीज में केवल एक बार करनी होगी।

इस परियोजना ने [Microsoft Open Source Code of Conduct](https://opensource.microsoft.com/codeofconduct/) को अपनाया है।
अधिक जानकारी के लिए देखें [Code of Conduct FAQ](https://opensource.microsoft.com/codeofconduct/faq/) या
