# AGENTS.md

## परियोजना अवलोकन

यह रिपॉजिटरी "शुरुआती के लिए AI एजेंट" को शामिल करता है - एक व्यापक शैक्षिक कोर्स जो AI एजेंट बनाने के लिए आवश्यक सभी कुछ सिखाता है। कोर्स में 15+ पाठ शामिल हैं जो मूल बातें, डिजाइन पैटर्न, फ्रेमवर्क और AI एजेंट्स के उत्पादन परिनियोजन को कवर करते हैं।

**मुख्य तकनीकें:**
- Python 3.12+
- इंटरैक्टिव सीखने के लिए Jupyter नोटबुक्स
- AI फ्रेमवर्क: Microsoft Agent Framework (MAF)
- Azure AI सेवाएं: Microsoft Foundry, Azure AI Foundry Agent Service V2

**आर्किटेक्चर:**
- पाठ-आधारित संरचना (00-15+ निर्देशिकाएँ)
- प्रत्येक पाठ में शामिल हैं: README दस्तावेज़, कोड उदाहरण (Jupyter नोटबुक्स), और चित्र
- स्वचालित अनुवाद प्रणाली के माध्यम से बहुभाषी समर्थन
- प्रत्येक पाठ के लिए एक Python नोटबुक जिसका उपयोग Microsoft Agent Framework करता है

## सेटअप कमांड्स

### आवश्यकताएँ
- Python 3.12 या उससे ऊपर
- Azure सब्सक्रिप्शन (Azure AI Foundry के लिए)
- Azure CLI इंस्टॉल और प्रमाणित (`az login`)

### प्रारंभिक सेटअप

1. **रिपॉजिटरी क्लोन या फोर्क करें:**
   ```bash
   gh repo fork microsoft/ai-agents-for-beginners --clone
   # या
   git clone https://github.com/microsoft/ai-agents-for-beginners.git
   cd ai-agents-for-beginners
   ```

2. **Python वर्चुअल एनवायरनमेंट बनाएँ और सक्रिय करें:**
   ```bash
   python3 -m venv venv
   source venv/bin/activate  # विंडोज़ पर: venv\Scripts\activate
   ```

3. **निर्भरता स्थापित करें:**
   ```bash
   pip install -r requirements.txt
   ```

4. **पर्यावरण चर सेट करें:**
   ```bash
   cp .env.example .env
   # अपने API कुंजी और एंडपॉइंट के साथ .env संपादित करें
   ```

### आवश्यक पर्यावरण चर

**Azure AI Foundry** के लिए (आवश्यक):
- `AZURE_AI_PROJECT_ENDPOINT` - Azure AI Foundry प्रोजेक्ट एंडपॉइंट
- `AZURE_AI_MODEL_DEPLOYMENT_NAME` - मॉडल डिप्लॉयमेंट नाम (जैसे gpt-4o)

**Azure AI Search** के लिए (पाठ 05 - RAG):
- `AZURE_SEARCH_SERVICE_ENDPOINT` - Azure AI Search एंडपॉइंट
- `AZURE_SEARCH_API_KEY` - Azure AI Search API कुंजी

प्रमाणीकरण: नोटबुक चलाने से पहले `az login` चलाएँ (जो `AzureCliCredential` का उपयोग करता है)।
