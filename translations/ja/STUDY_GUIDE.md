# 初心者向けAIエージェント - 学習ガイド＆コース概要

このガイドは「初心者向けAIエージェント」コースの概要を提供し、AIエージェント構築のための主要な概念、フレームワーク、デザインパターンを説明します。

## 1. AIエージェントのイントロダクション

**AIエージェントとは？**  
AIエージェントは、大規模言語モデル（LLM）の能力を拡張し、**ツール**、**知識**、**メモリ**へのアクセスを可能にするシステムです。単に学習データに基づいてテキストを生成する標準のLLMチャットボットとは異なり、AIエージェントは以下を行えます：  
- **環境を知覚する**（センサーや入力を通じて）。  
- **問題解決の方法を推論する**。  
- **環境を変えるために行動を起こす**（アクチュエーターやツール実行を通じて）。

**エージェントの主な構成要素：**  
- **環境**：エージェントが動作する空間（例：予約システム）。  
- **センサー**：情報収集の仕組み（例：APIの読み取り）。  
- **アクチュエーター**：行動を実行する仕組み（例：メールの送信）。  
- **ブレイン（LLM）**：計画を立ててどの行動をとるか決定する推論エンジン。  

## 2. エージェントフレームワーク

本コースでは、エージェント構築に**Microsoft Agent Framework (MAF)** と **Azure AI Foundry Agent Service V2** を使用します：

| コンポーネント | フォーカス | 最適な用途 |
|-----------|-------|----------|
| **Microsoft Agent Framework** | エージェント、ツール、ワークフローの統合Python/C# SDK | ツール搭載エージェント構築、マルチエージェントワークフロー、生産環境パターン。 |
| **Azure AI Foundry Agent Service** | マネージドクラウド実行環境 | 組み込みの状態管理、可観測性、信頼性を備えた安全でスケーラブルな展開。 |

## 3. エージェントデザインパターン

デザインパターンは、エージェントが信頼性高く問題を解決するための構造を支援します。

### **ツール利用パターン** (レッスン4)  
このパターンはエージェントが外部の世界とやり取りすることを可能にします。  
- **コンセプト**：エージェントに「スキーマ」（使用可能な関数とパラメータの一覧）が提供されます。LLMがユーザーの要望に基づいて*どの*ツールを呼び出し、*どの*引数を渡すかを決定します。  
- **フロー**：ユーザー要望 -> LLM -> **ツール選択** -> **ツール実行** -> LLM（ツール出力付き） -> 最終応答。  
- **使用例**：リアルタイムデータ取得（天気、株価）、計算実行、コードの実行。  

### **計画パターン** (レッスン7)  
このパターンはエージェントが複雑で多段階のタスクを解決できるようにします。  
- **コンセプト**：エージェントが高レベルの目標を一連の小さなサブタスクに分解します。  
- **アプローチ**：  
  - **タスク分解**：「旅行の計画」を「飛行機予約」「ホテル予約」「レンタカー予約」に分割。  
  - **反復計画**：前工程の結果に基づき計画を再評価（例：フライトが満席なら別の日を選択）。  
- **実装例**：構造化された計画（例：JSON）を生成する「プランナー」エージェントを用い、それを他のエージェントが実行。  

## 4. デザイン原則

エージェント設計には3つの視点を考慮します：  
- **空間**：エージェントは人と知識をつなぎ、アクセスしやすく目立たず存在すべき。  
- **時間**：エージェントは*過去*から学び、*現在*に適切な働きかけを行い、*未来*に順応すべき。  
- **コア**：不確実性を受け入れつつ、透明性とユーザー制御を通じて信頼を構築。  

## 5. 重要なレッスンのまとめ

- **レッスン1**：エージェントは単なるモデルではなく、知覚し、推論し、行動するシステム。  
- **レッスン2**：Microsoft Agent Frameworkはツール呼び出しや状態管理の複雑さを抽象化。  
- **レッスン3**：透明性とユーザー制御を念頭に置いて設計。  
- **レッスン4**：ツールはエージェントの「手」にあたり、スキーマ定義はLLMに使い方を理解させる鍵。  
- **レッスン7**：計画はエージェントの「実行機能」であり、複雑なワークフローの遂行を可能にする。
