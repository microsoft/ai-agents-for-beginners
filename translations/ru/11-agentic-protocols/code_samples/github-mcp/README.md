# Github MCP Server Example

## Описание

Это была демонстрация, созданная для AI Agents Hackathon, проводимого через Microsoft Reactor.

Инструмент используется для рекомендаций проектов хакатона на основе репозиториев пользователя в Github.
Это достигается следующим образом:

1. **Github Agent** - Использование Github MCP Server для получения репозиториев и информации о них.
2. **Hackathon Agent** - Берет данные от Github Agent и генерирует креативные идеи проектов для хакатона на основе проектов, языков, используемых пользователем, и направлений проектов для AI Agents hackathon.
3. **Events Agent** - На основе предложений hackathon agent, events agent рекомендует соответствующие мероприятия из серии AI Agent Hackathon.

## Запуск кода

### Переменные окружения

Демонстрация использует Microsoft Agent Framework, Azure OpenAI Service, Github MCP Server и Azure AI Search.

Убедитесь, что у вас установлены правильные переменные окружения для использования этих инструментов:

```python
AZURE_AI_PROJECT_ENDPOINT=""
AZURE_AI_MODEL_DEPLOYMENT_NAME=""
AZURE_SEARCH_SERVICE_ENDPOINT=""
AZURE_SEARCH_API_KEY=""
``` 


## Запуск Chainlit Server

Для подключения к MCP серверу, в этой демонстрации используется Chainlit в качестве интерфейса чата.

Чтобы запустить сервер, используйте следующую команду в терминале:

```bash
chainlit run app.py -w
```


Это должно запустить ваш Chainlit сервер на `localhost:8000`, а также заполнить ваш Azure AI Search Index содержимым `event-descriptions.md`.

## Подключение к MCP Server

Для подключения к Github MCP Server, выберите иконку "штекера" под полем чата "Type your message here..":

![MCP Connect](../../../../../translated_images/ru/mcp-chainlit-1.7ed66d648e3cfb28.webp)

Затем вы можете нажать на "Connect an MCP", чтобы добавить команду для подключения к Github MCP Server:

```bash
npx -y @modelcontextprotocol/server-github --env GITHUB_PERSONAL_ACCESS_TOKEN=[YOUR PERSONAL ACCESS TOKEN]
```


Замените "[YOUR PERSONAL ACCESS TOKEN]" на ваш реальный персональный токен доступа.

После подключения вы должны увидеть (1) рядом с иконкой штекера, подтверждающее подключение. Если нет, попробуйте перезапустить chainlit сервер с помощью `chainlit run app.py -w`.

## Использование демонстрации

Чтобы запустить рабочий процесс агентов для рекомендаций проектов хакатона, вы можете ввести сообщение, например:

"Recommend hackathon projects for the Github user koreyspace"

Router Agent проанализирует ваш запрос и определит, какое сочетание агентов (GitHub, Hackathon и Events) лучше всего подходит для обработки вашего запроса. Агенты работают вместе, чтобы предоставить комплексные рекомендации на основе анализа репозиториев GitHub, идей проектов и релевантных технических мероприятий.
