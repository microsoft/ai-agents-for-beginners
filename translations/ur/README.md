>
> **CMD (Windows):**
> ```cmd
> git clone --filter=blob:none --sparse https://github.com/microsoft/ai-agents-for-beginners.git

## 🌱 آغاز کریں

یہ کورس مصنوعی ذہانت کے ایجنٹس بنانے کے بنیادی اصولوں پر مبنی اسباق پر مشتمل ہے۔ ہر سبق اپنے موضوع پر مشتمل ہے، اس لیے جہاں چاہیں وہاں سے شروع کریں!

اس کورس کے لیے کثیراللسانی حمایت موجود ہے۔ یہاں ہمارے [دستیاب زبانوں کو دیکھیں](#-multi-language-support)۔

اگر یہ آپ کا پہلا موقع ہے کہ آپ Generative AI ماڈلز کے ساتھ کام کر رہے ہیں، تو ہمارا [Generative AI For Beginners](https://aka.ms/genai-beginners) کورس دیکھیں، جس میں GenAI کے ذریعے 21 اسباق شامل ہیں۔

यह मत भूलो کہ [اس رپو کو ستارہ (🌟) دیں](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) اور [اس رپو کو فورک کریں](https://github.com/microsoft/ai-agents-for-beginners/fork) تاکہ کوڈ چل سکے۔

### دوسرے سیکھنے والوں سے ملیں، اپنے سوالات پوچھیں

اگر آپ پھنس جائیں یا مصنوعی ذہانت کے ایجنٹس بنانے بارے میں کوئی سوال ہو، تو ہمارے مخصوص Discord چینل میں شامل ہوں جو [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord) میں موجود ہے۔

### جو چیزیں آپ کو چاہئیں

اس کورس کے ہر سبق میں کوڈ کے نمونے شامل ہیں، جو code_samples فولڈر میں مل سکتے ہیں۔ آپ [اس ریپو کو فورک کر سکتے ہیں](https://github.com/microsoft/ai-agents-for-beginners/fork) تاکہ اپنی کاپی بنا سکیں۔

ان مشقوں کے کوڈ مثالیں Microsoft Agent Framework اور Azure AI Foundry Agent Service V2 استعمال کرتی ہیں:

- [Microsoft Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Azure اکاؤنٹ ضروری ہے

یہ کورس Microsoft کے مندرجہ ذیل AI Agent frameworks اور خدمات استعمال کرتا ہے:

- [Microsoft Agent Framework (MAF)](https://aka.ms/ai-agents-beginners/agent-framework)
- [Azure AI Foundry Agent Service V2](https://aka.ms/ai-agents-beginners/ai-agent-service)

کچھ کوڈ نمونے متبادل OpenAI کی موافق فراہم کنندگان کی بھی حمایت کرتے ہیں جیسے [MiniMax](https://platform.minimaxi.com/)، جو بڑے سیاق و سباق کے ماڈلز فراہم کرتا ہے (204K ٹوکنز تک)۔ ترتیب کی تفصیلات کے لیے [Course Setup](./00-course-setup/README.md) دیکھیں۔

اس کورس کے کوڈ چلانے کے مزید معلومات کے لیے [Course Setup](./00-course-setup/README.md) پر جائیں۔
