## üå± Kuanzia

Kozi hii ina masomo yanayofunika misingi ya kujenga Wakala za AI. Kila somo linahusu mada yake hivyo anza mahali popote unapopenda!

Kuna msaada wa lugha nyingi kwa kozi hii. Nenda kwenye [lugha zinazopatikana hapa](../..). 

Ikiwa hapa ni mara yako ya kwanza kujenga kwa kutumia mifano ya AI Inayozalisha, angalia kozi yetu ya [AI Inayozalisha kwa Waanzilishi](https://aka.ms/genai-beginners), ambayo ina masomo 21 juu ya kujenga kwa GenAI.

Usisahau [kuweka nyota (üåü) kwenye repo hii](https://docs.github.com/en/get-started/exploring-projects-on-github/saving-repositories-with-stars?WT.mc_id=academic-105485-koreyst) na [kufork repo hii](https://github.com/microsoft/ai-agents-for-beginners/fork) ili kuendesha msimbo.

### Kutana na Wanafunzi Wengine, Pata Majibu ya Maswali Yako

Kama utakwama au una maswali yoyote kuhusu kujenga Wakala za AI, jiunge na kituo chetu maalum cha Discord katika [Microsoft Foundry Discord](https://aka.ms/ai-agents/discord).

### Unachohitaji 

Kila somo katika kozi hii lina mifano ya msimbo, ambayo inaweza kupatikana katika folda code_samples. Unaweza [kufork repo hii](https://github.com/microsoft/ai-agents-for-beginners/fork) ili kuunda nakala yako mwenyewe.  

Mfano wa msimbo katika mazoezi haya, unatumia Azure AI Foundry na GitHub Model Catalogs kwa ajili ya kuingiliana na Modeli za Lugha:

- [Github Models](https://aka.ms/ai-agents-beginners/github-models) - Bure / Imewekewa kikomo
- [Azure AI Foundry](https://aka.ms/ai-agents-beginners/ai-foundry) - Inahitaji Akaunti ya Azure

Kozi hii pia inatumia mifumo ifuatayo ya Wakala wa AI na huduma kutoka Microsoft:

- [Microsoft Agent Framework (MAF) - Mpya!](https://aka.ms/ai-agents-beginners/agent-framewrok)
- [Azure AI Agent Service](https://aka.ms/ai-agents-beginners/ai-agent-service)
- [Semantic Kernel](https://aka.ms/ai-agents-beginners/semantic-kernel)
- [AutoGen](https://aka.ms/ai-agents/autogen)


Kwa taarifa zaidi kuhusu jinsi ya kuendesha msimbo wa kozi hii, nenda kwenye [Usanidi wa Kozi](./00-course-setup/README.md).
